## 1. 理解索引、切片的概念，使用它们读取序列中的元素

**来源：05_内置组合数据类型**

### 字符串索引

- index，索引，又叫下标，指字符串中每个字符的位置编号。
- 通过位置编号，可以随机访问字符串中的任何一个字符。
- 字符串中的每个字符，都有二个位置编号（索引、下标、index）。
- 一种编号方式是使用非负整数，假设一个长度为n的字符串，从左向右分别编号为0，1，……，n-1。这种编号方式与大多数语言（如C语言）类似。特别要注意编号从0开始！
- 另一种编号方式是使用负整数，从右向左分别编号为-1，-2，……，-n。这种编号方式是大多数其它语言没有的。
- 使用一对方括号来访问某一个字符，格式：字符串[索引] 。

```
+---+---+---+---+---+---+
| p | y | t | h | o | n |
+---+---+---+---+---+---+
  0   1   2   3   4   5
 -6  -5  -4  -3  -2  -1
```

```python
s = 'python'
# 从左向右依次显示s中的每个字符
for c in s: 
    print(c)   
# 从左向右依次显示s中的每个字符    
for i in range(len(s)):
    print(s[i])
# 从右向左（反向）依次显示s中的每个字符    
for i in range(-1,-len(s)-1,-1):
    print(s[i])
```

### 字符串切片

- 指定编号的起止位置，可以从一个字符串中切出一段字符串，叫做切片。
- 格式：字符串[起点:终点:步长] ，如`s[m:n:p]`。步长p和第2个冒号可缺省，默认p=1。起点m和终点n也可以省略（其含义在下面介绍），但是注意第1个冒号不可省略，即使在省略了m或n的情况下。
- 注意：m和n中如果有负值索引，Python会自动转换成非负值索引再进行处理。
- 如果m和n越界，并不会导致错误。

设字符串s的长度len(s)=n，则常见的几种切片写法的含义说明如下。

- `s[m:n]`：取下标大于等于m但小于n的字符组成切片，即从下标m到n-1的字符。如果n<=m，则切片为空字符串。
- `s[m:]`：取下标大于等于m的字符组成切片，即从下标为m的字符到字符串s的结尾。
- `s[:n]`：取下标小于n的字符组成切片，即从下标0到n-1的字符。
- `s[:]`：取所有字符，即切片为字符串s的复本。
- `s[m:n:p]`
  - 当p为正值时，以间隔p取下标大于等于m但小于n的字符，即下标为m,m+p,m+2p,……的字符，且最后一个字符小于n。
  - 当p为负值时，以间隔p取下标小于等于m但大于n的字符，即下标为m,m+p,m+2p,……的字符，且最后一个字符大于n。
- `s[::-1]`：取字符串s反转所有字符顺序的后组成的字符串，即s的倒序。

```python
s = 'python'
print(s[1:4])   # yth
print(s[-4:4])  # th
print(s[-2:-4]) # 空字符串
print(s[3:])    # hon
print(s[:-3])   # pyt
print(s[5:10])  # n
print(s[1:5:2]) # yh
print(s[:])     # python
print(s[::-1])  # nohtyp
```

- 元组的索引方式和字符串一样。正向索引从 0 到 len()-1，反向索引从 -1 到 -len() 。
- 元组的切片和字符串也是一样的。[a:b] 取从索引为 a 到 b 之间（含a不含b）的元素。

```python
t = (1, -2, 0, 3, -4)
print(t[1])     # -2
print(t[1:4])   # (-2, 0, 3)
```

- 列表同样可以使用索引和切片，对列表中的部分元素进行截取。

## 2. 加法运算符（序列+序列），乘法运算符（序列*整数），成员运算符（in, not in），比较运算符（==, !=, <, <=, >, >=）

**来源：05_内置组合数据类型**

### 加法运算符

- 加法，形式为 `a+b` ，其中 a 和 b 都必须为相同类型序列，结果为两个序列相连。
- 字符串示例："hello"+"world" 将得到 "helloworld" ，注意两个字符串之间不会自动加空格。

```python
# 元组连接
a = (1, 3, 5)
b = (2, 4, 6)
print(a+b) # (1, 3, 5, 2, 4, 6)
```

- 列表与元组一样，可以使用加法运算符连接二个列表。但是要注意只有相同的类型才能连接，要么全是列表要么全是元组。

```python
>>> a = [1,2,3]
>>> b = (4,5,6)
>>> a+b
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: can only concatenate list (not "tuple") to list
>>> a+list(b)
[1, 2, 3, 4, 5, 6]
```

### 乘法运算符

- 乘法，形为`s*n 或 n*s`，其中s为序列，n为整数，结果为序列s重复n遍。
- 字符串示例："hello"*3 将得到 "hellohellohello"，注意多个字符串之间不会自动加空格。

```python
# 元组乘法
chest = ("gold", "gems")
print(t*3) # ('gold', 'gems', 'gold', 'gems', 'gold', 'gems')
```

### 成员运算符 in, not in

- in 操作符可以判断一个字符或字符串是否包含在另一个字符串当中。

```python
s = 'world'
print('wo' in s)    # True
print('wd' in s)    # False
print('e' in s)     # False
print('e' not in s) # True
```

- 元组使用 in 和 not in 运算测试元素成员性，通常用于创建条件。

```python
if 6 in t:
    print("t has a six.")
```

- 列表与元组一样，可以使用 in 和 not in 运算测试列表的元素成员性。

### 比较运算符

- 字符串的相等比较。使用==判断是否相等，使用!=判断是否不相等。相等的含义即两个字符串一模一样。
- 字符串不相等时比大小，规则是两个字符串的字符逐一对齐比较。字符比较大小的依据是它的Unicode编码，编码越大，字符在比较时就越大。如果两个字符串的前面部分一样，但是一个字符串较长，则较长者较大。

```python
>>> 'apple' < 'banana'  # True，因为双方的第一个字符 'a' < 'b'
True
>>> 'apple' < 'application' # True，因为双方的第五个字符 'e' < 'i'
True
>>> 'apple' < 'applepie' # True，因为双方前五个字符相同，但是后者更长
True
```

- 元组可以使用比较运算符 ==, !=, >, >=, <, <= 比较二个元组，比较方法是元素之间逐一比较。

```python
t1 = ('ant','apple','banana')
t2 = ('ant','applepie','avatar')
print(t1 < t2) # True
```

## 3. 函数：len(), max(), min(), sorted(), reversed(), sum()（除len()外，其它函数要求元素类型相同，sum()仅适合于数值型元素）

**来源：05_内置组合数据类型**

### len() 求长度

- len()可以返回字符串长度，即包含字符的个数，包括空格、换行等符号。注意一个汉字算一个字符。

```python
message = "Hello world!\n你好，世界！"
print(len(message)) # 19
```

- 使用 len() 函数可以求出元组长度
- 使用 len() 可以求列表的长度

### max(), min() 找最大最小值

- max(string) 返回字符串中编码最大的字母
- min(string) 返回字符串中编码最小的字母

```python
s = 'world'
print(max(s),min(s)) # w d
```

- 使用 max(), min() 函数可以求出元组中的最大元素和最小元素，前提是元素的类型是可比较的。
- 同样适用于列表。

### sum() 求和

- 使用 sum() 函数可以求元组所有元素之和，前提是元素的类型是可相加的。

```python
t = (1, -2, 0, 3, -4)
print(sum(t)) # -2
```

- sum()同样适用于列表，但不适用于字符串。

### sorted() 排序

- 使用 sorted(iterable) 函数，也可以将列表中的内容排序。内建函数 sorted 不是列表的内置函数，可以对所有可迭代的对象 iterable 进行排序操作。sorted 不会修改列表本身，而是返回一个新的排序后的列表。
- 可选的参数 reverse ：排序时默认为升序，加上 reverse=True 可改为降序。
- 可选的参数 key ：使用 key 参数接收一个函数来指定排序的逻辑。

```python
list1 = [5,3,7]
list2 = sorted(list1) # list1内容不变，排序结果放list2中
print(list1) # [5, 3, 7]
print(list2) # [3, 5, 7]
list2 = sorted(list1,reverse=True)
print(list2) # [7, 5, 3]

# 按照字符串长度而非字典顺序进行排序
d = sorted(['apple','ant','duck'],key=lambda s:len(s))
print(d) # ['ant', 'duck', 'apple']
```

- sorted()同样适用于字符串和元组。

### reversed() 翻转

- 使用 reversed(list) 函数，可以将列表中的内容顺序翻转。这个函数不会修改列表本身。

```python
list1 = [3,5,7]
list2 = list(reversed(list1)) # list1内容不变，翻转结果放list2中
print(list1) # [3, 5, 7]
print(list2) # [7, 5, 3]
```

- reversed()同样适用于字符串和元组。

### 通用函数总结表（来源：05_内置组合数据类型）

|      方法       |   作用   | 是否适用于字符串 | 是否适用于元组 | 是否适用于列表 |
| :-------------: | :------: | :--------------: | :------------: | :------------: |
|   len(items)    |  求长度  |        是        |       是       |       是       |
|   max(items)    | 求最大值 |        是        |       是       |       是       |
|   min(items)    | 求最小值 |        是        |       是       |       是       |
|   sum(items)    |   求和   |        否        |       是       |       是       |
| reversed(items) |   翻转   |        是        |       是       |       是       |
|  sorted(items)  |   排序   |        是        |       是       |       是       |
|      map()      |   映射   |        是        |       是       |       是       |
|      zip()      |   匹配   |        是        |       是       |       是       |

## 4. 内置函数：index(), count()

**来源：05_内置组合数据类型**

### index() 查找

- tuple.index(x[, start[, end]])：返回元组中第一个值为 *x* 的元素的零基索引。未找到指定元素时，触发 ValueError 异常。可选参数 start 和 end 是切片符号，用于将搜索限制为元组的特定子序列。

```python
fruits = ('orange', 'apple', 'pear', 'banana', 'kiwi', 'apple', 'banana')
i = fruits.index('banana')
# 3
i = fruits.index('banana', i+1)  # Find next banana starting at position 4
# 6
```

### count() 计数

- tuple.count(x)：返回元组中元素 x 出现的次数。

```python
fruits = ('orange', 'apple', 'pear', 'banana', 'kiwi', 'apple', 'banana')
a = fruits.count('apple')
# 2
b = fruits.count('tangerine')
# 0
```

- 列表与元组一样，可以使用内置函数 index() 和 count() 完成查找和计数。

### 元组或列表的内置函数总结

|              方法              |   作用   | 是否适用于列表 | 是否适用于元组 |
| :----------------------------: | :------: | :------------: | :------------: |
| index(x[, start[, end]]) |   查找   |       是       |       是       |
|            count(x)            | 统计个数 |       是       |       是       |

## 1. 函数：tuple()

**来源：05_内置组合数据类型**

- 创建元组，可以直接用带小括号的序列形式赋值，也可以使用tuple()函数。

```python
t = tuple()                     # () 空元组
t = tuple(1,2,3)                # 错误！
t = tuple(1)                    # 错误！
t = (1)                         # 语法上没有错误，但是t此时的值为一个整数，而不是元组
t = (1,)                        # (1,) 此时t为仅有一个元素的元组
```

## 1. 使用索引和切片替换列表中的元素

**来源：05_内置组合数据类型**

### 通过索引对列表元素进行赋值

- 因为列表可变，可以直接给现有元素赋新值。
- 虽然索引方法可以修改列表中的元素，但是不能创建新元素，也不能修改不存在的元素。

```python
a = [1, 3, 5, 7, 9]
a[1] = 6
print(a)
# [1, 6, 5, 7, 9]
```

### 通过切片对列表进行赋值

- 下面赋值语句会修改列表。因为将只有一个元素的列表赋值给了拥有两个元素的切片，所以列表的长度就会缩短1个单位

```python
a = [1, 3, 5, 7, 9]
a[1:3] = [6]
print(a)
# [1, 6, 7, 9]
```

## 2. 函数：list(), sorted(), reversed()

**来源：05_内置组合数据类型**

### list() 创建列表

```python
# 使用字面量直接为列表赋值
list1 = [] # 空列表
list2 = [1] # 与元组不同，一个元素也不必在最后加逗号，因为不会混淆
list3 = [2,3,4]

# 使用list()函数创建列表
list1 = list() # 空列表
```

```python
list1 = [5,3,7]
```

## 3. 内置函数：append(), insert(), extend(), pop(), remove(), clear(), reverse(), sort()

**来源：05_内置组合数据类型**

### append(x) 追加

```python
list1 = [1,2,3]
list1.append(6)
print(list1)
# [1, 2, 3, 6]
```

- 使用 list.append(x) 将元素 x 添加到列表末尾，列表长度加一。

### insert(i, x) 插入

```python
list1.insert(1,8)
print(list1)
# [1, 8, 2, 3, 6]
```

- 使用 list.insert(i, x) 将元素 x 添加到列表中索引为 i 的位置，原来索引为 i的及其后面的所有元素向后移一位，列表长度加一。

### extend(iterable) 扩展

```python
list2 = [5,7,9]
list1.extend(list2)
print(list1)
# [1, 8, 2, 3, 6, 5, 7, 9]
```

- 使用 list.extend(iterable)，将可迭代对象的元素添加到 list 列表末尾。

### remove(x) 删除

```python
list1 = [1,2,3,8,4,5,6,8,7]
if 9 in list1:
    list1.remove(9) # 如果没有判断是否存在就删除，此处就会报错。
while 8 in list1:
    list1.remove(8) # 配合循环才能把某个内容的元素删除干净
print(list1) # [1, 2, 3, 4, 5, 6, 7]
```

- 使用 list.remove(x) 删除列表中内容为x的元素。注意如果x不存在，则会出错，如果内容为x的元素不只一个，则只会删除第一个。

### pop([i]) 弹出

```python
list1 = [1,2,3,8,4,5,6,8,7]
x = list1.pop()     # x = 7
y = list1.pop(3)    # y = 8
print(list1)        # [1, 2, 3, 4, 5, 6, 8]
```

- 使用 x = list.pop([i]) ，可以删除列表中索引为 i 的元素，并且被删除的元素会放入变量 x 中。如果不指定 i ，则删除列表中最后一个元素。

### clear() 清空

- 使用 list.clear() 函数，可以删除列表里的所有元素。

### reverse() 翻转

```python

# [7, 5, 3]
```

- 使用 list.reverse() 函数，可以将列表中的内容顺序翻转。这个函数将修改列表本身。

### sort() 排序

```python
list1 = [5,3,7]
list1.sort()
print(list1) # list1的内容直接排序
# [3, 5, 7]
list1.sort(reverse=True)
print(list1)
# [7, 5, 3]
```

- 使用 list.sort() 函数，可以将列表中的内容排序。这个函数将修改列表本身，函数无返回值。
- 可选的参数 reverse ：排序时默认为升序，加上 reverse=True 可改为降序。
- 可选的参数 key ：使用 key 参数接收一个函数来指定排序的逻辑。

## 5. 列表生成式（简单情况：一重循环，带条件）

**来源：05_内置组合数据类型**

- 推导式 comprehensions（又称解析式），是Python的一种独有特性，可以从一个数据序列构建另一个新的数据序列。
- 列表推导式的语法形式为：

```python
[expression for expr1 in sequence1 if condition1
for expr2 in sequence2 if condition2
...
for exprN in sequenceN if conditionN]
```

- 列表推导式在逻辑上等价于一个循环语句，只是形式上更加简洁。

```python
# 列表推导式
alist = [ x*x for x in range(10) ]
print(alist)
# [0, 1, 4, 9, 16, 25, 36, 49, 64, 81]

# 等价循环
alist = []
for x in range(10):
    alist.append(x*x)
print(alist)
# [0, 1, 4, 9, 16, 25, 36, 49, 64, 81]
```

### 过滤不符合条件的元素（一重循环+条件）

```python
alist = [-1, -2, 1, 2, 3, -4, -5, -6]
blist = [ i for i in alist if i>0 ]
print(blist)
# [1, 2, 3]
```

### 更多例子

```python
# 知道有一个包含一些同学成绩的字典，现在需要计算所有成绩的最高分、最低分、平均分，并查找所有最高分的同学
scores = {'Zhang': 45, 'Li': 78, 'Wang': 40, 'Zhou': 96, 'Zhao': 65, 'Sun': 90}
highest = max(scores.values())
lowest = min(scores.values())
average = sum(scores.values())/len(scores)
highest_person = [ name for name,score in scores.items() if score == highest ]
print (highest,lowest,average,highest_person)
# 96 40 69.0 ['Zhou']
```

```python
# 查找列表中最大元素的所有位置
from random import randint
x = [ randint(1,10) for i in range(10) ]
print(x) # [9, 1, 9, 1, 2, 8, 10, 10, 1, 6]
m=max(x)
print(m) # 10
max_index = [ index for index,value in enumerate(x) if value == m ]
```

## 6. 列表、字典、集合的可变性

**来源：05_内置组合数据类型**

- 以下数据类型属于**不可变数据类型：字符串、元组**。
- 以下数据类型属于**可变数据类型：列表、字典、集合**。

### 列表的可变性

- 列表和元组最大的区别就是：列表是可变的，可以修改。
- 这里的可变性，或者说是可修改，指的是可以通过索引、切片、列表方法等方式，向列表中增加、删除或替换部分元素。
- 注意，通过向元组变量或列表变量直接赋值的方法是整体替换，不是部分修改，不受可变性约束。

```python
t = (1,3,5)
print(t[0]) # 1
t[0] = 7        # 错误！元组具有不可变性。
t = (7,8,9)     # 正确，直接向元组变量赋值整体替换一个元组是可行的。

l = [2,4,6]
print(l[0]) # 2
l[0] = 7        # 正确，列表具有可变性
l = [7,8,9]     # 正确，直接向列表变量赋值整体替换一个列表也是可行的。
```

### 字符串的不可变性

- 字符串的不可变性，不是指字符串不可以赋新值。
- 字符串可以整体赋新值，但是不可以通过索引和切片修改原字符串中的部分字符，这叫做字符串的不可变性。

```python
s = 'python'
r = ''
for c in s:
    r = c + r  # 正确
print(r) # nohtyp
for i in range(len(s)//2):
    s[i] = s[-i-1] # 错误
```

### 新增字典中的键值对

```python
student_no = input("Who do you want me to add (No.)? ")
```

## 1. 函数：set(), len(), max(), min(), sum()

### set() 创建集合

- Python同时使用花括号创建字典和集合，而 {} 被用来表示一个空字典。为了创建一个空集合，只能使用 set() 。

```python
s5 = set('python') # 从字符串生成集合
# {'t', 'h', 'o', 'y', 'n', 'p'}
```

### len(), max(), min(), sum()

```python
>>> s1 = {1, 3, 5}
>>> s1.add(6)
>>> s1
{1, 3, 5, 6}
>>> len(s1)
4
>>> max(s1)
6
>>> min(s1)
1
>>> sum(s1)
15
```

- len(), max(), min(), sum() 同样适用于集合。
