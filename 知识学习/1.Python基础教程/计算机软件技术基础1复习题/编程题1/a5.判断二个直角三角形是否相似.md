# 题目名称：判断二个直角三角形是否相似

## 题目描述

判断二个直角三角形是否相似。如果第一个直角三角形的两个直角边（a,b）比值等于另外一个直角三角形两个直角边（c,d）的比值，则视为相似。注意两条边的输入顺序不能影响结果的判定，即判断条件为：

$ a : b = c : d$  或者 $ a : b = d : c$ 

提示：为计算准确，能用乘法就不要用除法。判断条件转化为：

$ a \cdot d = b \cdot c $ 或者 $ a \cdot c = b \cdot d $


## 输入格式

先后四次输入正整数，前两个为第一个直角三角形的两个直角边，后两个为第二个直角三角形的直角边。

## 输出格式

如果两个直角三角形相似，则输出 True ，否则输出 False 。

## 输入输出样例

### 输入 #1
```
1
2
3
6
```

### 输出 #1
```
True 
```

### 输入 #2

```
1
2
6
3
```

### 输出 #2

```
True 
```

### 输入 #3

```
1
2
3
4
```

### 输出 #3

```
False
```
