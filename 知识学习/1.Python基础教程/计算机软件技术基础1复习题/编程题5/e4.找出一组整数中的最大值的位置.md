# 题目名称：找出一组整数中的最大值的位置

## 题目描述

读取若干个（至少1个）整数，找出它们中的最大值，并找到它的出现的位置(第1个整数的位置为1，后面依次增加）。

## 输入格式

第一个数字是样本数据的个数，然后后面是一组样本数据，一个一行。

## 输出格式

输入的数字中的最大值，然后是该最大值出现的位置，一个一行。

## 输入输出样例

### 输入 #1
```
7
5
8
2
4
8
2
8
```
### 输出 #1
```
8
2
5
7 
```

### 输入 #2
```
4
321
-58
89
321
```
### 输出 #2
```
321
1
4
```
