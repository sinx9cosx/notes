# 题目名称：春游报数

## 问题描述

老师带领同学们春游。已知班上有 N 位同学，每位同学有从 1 到 N 的唯一编号。到了集合时间，老师确认是否所有同学都到达了集合地点，就让同学们报出自己的编号。到达的同学都会报出，但有的同学很顽皮，会多次报出。你能帮老师找出有哪些同学没有到达吗？

## 输入描述

输入包含 2 行。第一行包含两个整数 N 和 M，表示班级有 N 位同学，同学们共有M次报出编号。约定 2≤N,M≤1000。

第二行包含 M 个整数，分别为 M 次报出的编号。约定所有编号都在合理范围内。

## 输出描述

输出一行。如果所有同学都到达，则输出 0；否则由小到大输出所有未到达的同学编号，空格分隔。

## 输入输出样例 #1

### 输入 #1
```
3 3
3 2 1
```
### 输出 #1
```
0
```
## 输入输出样例 #2

### 输入 #2
```
3 5
1 1 1 1 1
```
### 输出 #2
```
2 3
```
