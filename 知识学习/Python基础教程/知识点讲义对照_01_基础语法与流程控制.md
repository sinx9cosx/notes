## 1. 输入输出函数：print( end=, sep= ), input()

**来源：02_基础知识**

```python
print("Hello world!")
```

- print()用来输出信息
- 字符串：用一对引号包围起来的字符序列。一对引号，可以是一对单引号，也可以是一对双引号，但不可混用。

```python
name = input("What's your name? ")
print("Hello",name)
```

- input() 首先给出提示信息，然后等待用户输入。输入的信息为字符串类型，保存在赋值号前的变量中。
- print()函数会在输出的多个参数之间自动添加一个空格。

```python
print("Hello","world") 
Hello world
```

- 对比：二个字符串可以用+连接在一起，此时不会在中间自动添加一个空格。如"hello"+"world" => "helloworld"

### print() 的 sep 和 end 参数

```python
print('hello','world','!')
# hello world !
print('hello','world','!',sep='*')
# hello*world*!
print('hello','world','!',sep='')
# helloworld!
```

- 默认情况下，多个输出的对象之间，用一个空格分隔。可以用参数 sep 设置输出时对象之间的分隔符。
- 默认情况下，每次输出以换行符结尾。可以用参数 end 设置输出以何字符结尾。
- 参数 end 与 sep 的出现顺序没有要求，但是必须放在要显示的内容之后。
- 单独一个print()没有任何参数，会输出一个空换行。

```python
print('hello',end=' ')
print('world',end='')
print(' ',end='!')
print()
print('New line')

# hello world !
# New line
```

## 2. 合法标识符（变量名、函数名等）：只能含有数字, 字母以及下划线，不能以数字开头

**来源：02_基础知识**

- 标识符：作为变量、函数等名字的有效字符串集合（程序员取的名字，可以做变量名）
- 关键字（keyword）：Python语言保留使用的标识符，又称保留字，不能用于其它用途。（只有Python解释器能用，不可以做变量名）

```python
import keyword
print(keyword.kwlist)
['False', 'None', 'True', 'and', 'as', 'assert', 'async', 'await', 'break', 'class', 'continue', 'def', 'del', 'elif', 'else', 'except', 'finally', 'for', 'from', 'global', 'if', 'import', 'in', 'is', 'lambda', 'nonlocal', 'not', 'or', 'pass', 'raise', 'return', 'try', 'while', 'with', 'yield']
```

- 变量名只能含有数字, 字母以及下划线
- 变量名不能以数字开头
- 选择具有描述性的名字，做到望名识义！例如"score"而不是"s"。
- 风格保持一致，常见的变量命名风格有：
  - high_score（下划线命名法）
  - highScore（小驼峰命名法）
  - HighScore（大驼峰命名法）
- Python推荐下划线命名法。
- 大驼峰命名法一般不得用于变量命名（大驼峰命名法另有用途，如类名或为异常命名）
- 一般不得使用全大写字母做变量名（如：NUMBER_OF_THE_RESULT），这种命名一般代表常量
- 不用下划线开头或结尾（下划线开头或结尾的变量往往有特殊的用途）
- 变量名不要太长，最好在15个字符以内
- 除特殊情况外（如循环控制变量习惯用i），一般不得使用单字母作为变量名
- 一般使用英文单词命名，不允许英拼混用
- 不允许在变量名中出现汉字

## 3. 基本数据类型：int, float, string, bool

**来源：02_基础知识**

- Python 中数据类型，主要分为基本数据类型、内置容器类型和自定义类型
  - 基本数据类型：整数int(integer)、浮点数float、复数complex、布尔型bool(boolean)等
  - 内置容器类型：字符串型str(string)、元组tuple、列表list、字典dict(dictionary)、集合set等
  - 自定义类型：自定义类、第三方类等

### 整数

- 整数：没有小数部分的数字，例如：1, 27, -100, 0 . Python中的整数即数学中的整数，理论上 Python 可以处理任意大小的整数，包括正整数、零和负整数。

### 浮点数

- 浮点数：带有小数点的数字，例如：2.376, -99.1, 1.0 . Python中的浮点数并不等价于数学中的小数，Python 并不能处理所有的小数。
- 浮点数大小有限制。绝对值最大的浮点数为 1.7976931348623157e+308，绝对值最小的浮点数为 2.2250738585072014e-308。
- 浮点数精度有限制。浮点数只有15位有效数字，超长的部分没有意义。
- 整数和浮点数在计算机内部存储的方式是不同的，整数运算永远是精确的，而浮点数运算则可能会有误差。

> 著名的 0.1+0.2不等于0.3问题
> 0.1 + 0.2 = 0.30000000000000004

### bool 类型

- 二值逻辑：True、False
- 数字 0、0.0、0j 被视同 False，其它数字被视同 True
- 空字符串、空元组、空列表、空字典、空集合，视同 False，上述数据类型的非空值视同 True
- None被视同False
- 可以使用 bool() 将其它数据类型转换为布尔类型值。

```python
print(bool(0))      # False
print(bool(1))      # True
print(bool(''))     # False
print(bool('False'))    # True
```

## 4. 数值型的四则运算符：+, -, *, /, //, %, **

**来源：02_基础知识**

| 运算符 | 描述                                                            | 举例                   |
| :----- | :-------------------------------------------------------------- | :--------------------- |
| +      | 一元运算符，一元加，+a 即得到该操作数本身不变                   | 如a=-10，则+a仍然是-10 |
| -      | 一元运算符，一元减，-a 即得到变换该操作数的符号的数             | 如a=-10，则-a是10      |
| +      | 二元运算符，加法，a+b 两个对象相加                              | 10 + 20 结果 30        |
| -      | 二元运算符，减法，a-b 两个对象相减                              | 10 - 20 结果 -10       |
| *      | 二元运算符，乘法，a*b 两个对象相乘。                            | 10 * 20 结果 200       |
| /      | 二元运算符，算术除法（truediv），a/b 结果总是浮点数。           | 15 / 2 结果 7.5        |
| //     | 二元运算符，地板除法（floordiv），a//b 返回除法的商。           | 10 // 3 结果是 3       |
| %      | 二元运算符，取模运算（mod），又叫取余运算，a%b 返回除法的余数。 | 10 % 3 结果是 1        |
| **     | 二元运算符，乘方运算，又叫幂运算，a**b 返回a的b次幂             | 2 ** 3 结果是 8        |

- Python 不认识×号，所以乘法运算用* 代替。
- Python 不认识÷号，所以除法运算用/代替。

### 算术运算符的优先级（从高到低）

1. 幂运算符
2. 一元运算符：一元加、一元减
3. 乘除运算符：乘法、算术除法、地板除法、取模（取余）
4. 加减运算符：二元加、二元减

### 关于除法的说明

- 算术除法（truediv），算术除法会得到较精确的结果，结果总是浮点数，即使结果应为整数，也要在其后加上.0转为浮点数的形式。

```python
print(5/2) # 2.5
print(6/2) # 3.0
print(10/3) # 3.3333333333333335
```

- 地板除法（floordiv），地板除法得到算术除法结果的地板数，即除法的商。如果两个操作数中有浮点数，则结果以浮点数形式表示，即加上.0

```python
print(10//3)    # 3
print(9//3)     # 3
print(-9//3)    # -3
print(-10//3)   # -4
print(10//3.5)  # 2.0
print(-10//3.5) # -3.0
```

- 取余（取模）运算，得到除法的余数。具体的公式如下：设 x // y = z，则 x % y = x – (y*z)

```python
print(10%3)     # 1
print(-10%3)    # 2
print(-10%-3)   # -1
print(-10%-3.5) # -3.0
```

## 5. 常用内置数学函数：abs(), max(), min(), pow(), round()

**来源：02_基础知识**

| 函数           | 描述                                                           | 举例                |
| -------------- | -------------------------------------------------------------- | ------------------- |
| abs(x)         | 返回x的绝对值                                                  | abs(-2)=2           |
| max(x1,x2,...) | 返回x1,x2,...的最大值                                          | max(1,5,2)=5        |
| min(x1,x2,...) | 返回x1,x2,...的最小值                                          | min(1,5,2)=1        |
| pow(a,b)       | 返回a的b次方值，等价于a**b                                     | pow(2,3)=8          |
| round(x)       | 返回与x最接近的整数。如果x与两个整数接近程度相同，则返回偶数值 | round(4.5)=4        |
| round(x,n)     | 保留浮点数点后n位小数的浮点数值                                | round(5.466,2)=5.47 |

- round(x)函数与四舍五入的比较。一般情况下round(x)相当于数学上的将x四舍五入到整数，但是当x的小数部分恰好为.5时，即x与上下两个整数接近程度相同，则返回两个整数中的偶数值。如round(3.5)=4; round(4.5)=4; round(5.5)=6; round(-2.5)=-2; round(-3.5)=-4
- 对比：int(x)是将x的小数部分直接截断丢弃。如 int(4.5)=4; int(-3.5)=-3
- 由于浮点数的不精确性，当使用round(x,n)函数的参数n>0时，遇到被保留位后一位是5并且之后没有更多数字时，所得结果往往<mark style="background: #FF5582A6;">不可预测</mark>。如：round(5.215,2)=5.21; round(5.225,2)=5.22；还有：round(1.235,2)=1.24; round(1.335,2)=1.33

## 7. 逻辑运算符：not, and, or

**来源：02_基础知识**

- 三种运算：逻辑非 not、逻辑与 and、逻辑或 or
- 运算优先级：not > and > or

### 逻辑非 not

- 一元运算符，表示"相反"。not True即为False，not False即为True。

| 操作数A | not A |
| ------- | ----- |
| True    | False |
| False   | True  |

- 逻辑非运算的返回结果：只有True和False二值

```python
print(not 1)    # False
print(not '')   # True
```

### 逻辑与 and

- 二元运算符，表示"并且"，要求二个操作数均为 True 时，结果为 True，其它情况结果均为 False。

| 操作数A | 操作数B | A and B |
| ------- | ------- | ------- |
| True    | True    | True    |
| True    | False   | False   |
| False   | True    | False   |
| False   | False   | False   |

- 短路运算规则：如果已经判定第一个操作数是 False，则不再判断第二个操作数。
- 逻辑与运算的返回结果：最早确定表达式值的操作数。即，表达式 a and b 的返回值，如果 a 是 False，则返回 a，如果 a 是 True，则返回 b。

```python
print(6 and 'lucky')    # 'lucky'
print(0 and '')         # 0
```

### 逻辑或 or

- 二元运算符，表示"或者"，要求二个操作数均为 False 时，结果为 False，其它情况结果均为 True。

| 操作数A | 操作数B | A or B |
| ------- | ------- | ------ |
| True    | True    | True   |
| True    | False   | True   |
| False   | True    | True   |
| False   | False   | False  |

- 短路运算规则：如果已经判定第一个操作数是 True，则不再判断第二个操作数。
- 逻辑或运算的返回结果：最早确定表达式值的操作数。即，表达式 a or b 的返回值，如果 a 是 True，则返回 a，如果 a 是 False，则返回b。

```python
print(6 or 'lucky') # 6
print(0 or '')      # ''
```

## 8. 类型转换函数：int(), float(), str(), bool()

**来源：02_基础知识**

### 整数与浮点数相互类型转换

- 浮点数转整数：int(x)，当x为浮点数时，将x的小数部分直接截断丢弃转为整数。如：int(4.5)=4; int(-3.5)=-3
- 整数转浮点数：float(x)，当x为整数时，在x的后面补上.0转为浮点数。如：float(4)=4.0

### 浮点数转换为整数的几个函数的比较

- int(x)：当x为浮点数时，将x的小数部分直接截断丢弃转为整数。
- math.floor(x)：x向下取最近的整数，即不大于x的最大整数。
- math.ceil(x)：x向上取最近的整数，即不小于x的最小整数。
- round(x)：返回与x最接近的整数。如果x与两个整数接近程度相同，则返回偶数值。

### 字符串转数字

- int(x)：可将形为整数的字符串x转为整数，如int("45")=45. 注意x不可为形为浮点数的字符串，如：int("4.5")是会出错误的。
- float(x)：可将形为浮点数或整数的字符串x转为浮点数，如：float("45")=45.0; float("45.6")=45.6
- eval(x)：可以将字符串x视为一个表达式，计算其值。如：eval("1+2")=3

### 数字转字符串

- str(x)：可将整数或浮点数 x 转为字符串。

```python
a=2
b=3
s="The result of "+str(a)+"+"+str(b)+" is "+str(a+b)
print(s)
```

### bool()转换

```python
print(bool(0))      # False
print(bool(1))      # True
print(bool(''))     # False
print(bool('False'))    # True
```

```python
k = 1
```

## 10. 注释的写法：单行#，多行使用三引号

**来源：02_基础知识**

### 单行注释

- Python 使用 # 号实现单行注释。# 注释可以在一行中从头开始，也可以从中间开始。
- 从 # 开始，直到这行结束为止的所有内容都是注释。Python 解释器遇到 `#`时，会忽略它后面的整行内容。

```python
#使用print输出字符串
print("Hello World!")
print("Python is fun.")

#使用 print输出数字
print(100)
print( 3 + 100 * 2)
print( (3 + 100) * 2 )

print("Hello World!")   #输出Python欢迎信息
print( 36.7 * 14.5 )    #输出乘积
print( 100 % 7 )        #输出余数
```

### 多行注释

- 多行注释指的是一次性注释程序中多行的内容（包含一行）。
- Python 使用三个连续的单引号'''或者三个连续的双引号"""注释多行内容。
- 多行注释通常用来为 Python 文件、模块、类或者函数等添加版权或者功能描述信息。

```python
'''
使用 3 个单引号分别作为注释的开头和结尾
可以一次性注释多行内容
这里面的内容全部是注释内容
'''

"""
使用 3 个双引号分别作为注释的开头和结尾
可以一次性注释多行内容
这里面的内容全部是注释内容
"""
```

- Python 多行注释不支持嵌套。
- 不管是多行注释还是单行注释，当注释符作为字符串的一部分出现时，就不能再将它们视为注释标记，而应该看做正常代码的一部分。

## 11. if ... elif ... else 语句

- if语句的三种形式：if、if...else、if...elif...else
- 缩进：同一个代码段层次，需要相同的缩进，一般为4个空格。Python用缩进表达代码段的范围。
- pass语句：空语句，什么也不做，用于避免程序逻辑错误。

```python
cond1 = cond2 = False
if cond1:
    pass

if cond1:
    pass
else:
    pass

if cond1:
    pass
elif cond2:
    pass
else:
    pass
```

### 多分支if语句示例（成绩等级）

```python
score = eval(input("Enter the score: "))
if score >= 90:     # 90 <= score <= 100
    grade = 'A'
elif score >= 80:   # 80 <= score < 90
    grade = 'B'
elif score >= 70:   # 70 <= score < 80

    grade = 'D'
else:               # 0 <= score < 60
```

```python
    grade = 'C'

    grade = 'B'
```

## 13. while ... 语句

**来源：03_流程控制**

- 遇到while语句时，如果条件成立，while执行循环体中的语句
- 执行完每一轮循环后，while会重新判断条件，如果条件仍然成立，则再次执行循环体中的语句

```python
cond1 = True
while cond1:
    pass
```

- 在执行到while语句时，条件如果为False，则直接跳过循环，执行循环之后的语句。
- 在循环体中，必须有使条件判断结果发生变化的语句，否则循环将永远执行下去（称为死循环）。

### 例子：从1累加到100求和

```python
i=1
sum=0
while i<=100:
    sum += i
    i += 1
print(sum)
```

## 14. for ... in ... 语句

**来源：03_流程控制**

- for...in语句一般用于实现已经循环次数的循环。
- 基本结构：for 变量 in 序列，循环执行的次数与序列中元素的个数相同，每次循环中，变量自动赋值为序列中的某个元素。

```python
for c in 'hello':   # 循环5次，依次输出 h, e, l, l, o 五个字母
    print(c)
    
for i in (2,3,5,7): # 循环4次，依次输出 2, 3, 5, 7 四个数字
    print(i)
```

### while和for循环的转换

- 一般来说，已知循环次数的情况，可以用for循环，也可以用while循环，且for循环往往比较简单。
- 未知循环次数的情况，一般使用while循环。

```python
# while 循环
i = 1
while i < n:
    pass
    i += 1
    
# for 循环
for i in range(1,n):
    pass
```

## 15. break, continue, 用于循环中的 else 语句

**来源：03_流程控制**

### break

- break语句可以跳出 for 和 while 的循环体

### continue

- continue语句跳出当前轮循环，直接进行下一轮循环

```python
for i in range(1,101):
    if i % 7 == 0:

    if i % 10 == 7:
        continue
```

### 循环中的else

- else如果与while或for语句配合使用，意思是当循环正常退出（即不是遇到break语句而退出）时，就会执行else子句。
- 此语法为Python特有，其它语言（如C语言）一般没有将else用于循环语句中的语法。

```python
number = int(input())
for i in range(2,number):
    if number % i == 0:
        break
else:
    print(number,"is a prime number")
```

## 16. range() 函数：range(n), range(a,b), range(a,b,k)

**来源：03_流程控制**

- range()函数经常用在for...in语句中

- range(n)：产生一个 0,1,2,3,....,n-1 这n个数的序列，注意是从0开始，总数为n个。
  - 如果 n<= 0，生成的序列为空。
  - 例：range(5)：产生 0,1,2,3,4 这 5 个数组成的序列

- range(a,b)：产生从 a 到 b 的数字序列，注意是大于等于a且小于b，即包含 a 而不包含 b ！
  - 如果b<=a，生成的序列为空。
  - 例：range(1,10)：产生 1,2,3,...,9 这 9 个数的序列

- range(a,b,k)函数：产生一个a,a+k,a+k+k,...序列，即从 a 开始向 b 行进，但不能达到 b，即：
  - 如果 k>0，生成序列中的最大值不能大于或等于b。如果b<=a，生成的序列为空。
  - 如果 k<0，生成序列中的最大值不能小于或等于b。如果b>=a，生成的序列为空。
  - k不能等于0。省略参数 k，相当于 k=1
  - 例：range(0,10,2)，将产生0,2,4,6,8这个序列
  - 例：range(10,1,-1)，将产生10,9,8,7,6,5,4,3,2这个序列

## 19. time库：time()

**来源：02_基础知识 + 07_数值处理与计算**

### 来源：02_基础知识

- time.time() 获得现在距离 Unix Epoch（1970-01-01 00:00:00 UTC）所经过的时间，整数部分单位为秒，小数部分代表毫秒。

```python
import time

currentTime = time.time() # Get current time

# Obtain the total seconds since midnight, Jan 1, 1970
totalSeconds = int(currentTime)

# Get the current second 
currentSecond = totalSeconds % 60 

# Obtain the total minutes
totalMinutes = totalSeconds // 60 

# Compute the current minute in the hour
currentMinute = totalMinutes % 60

# Obtain the total hours
totalHours = totalMinutes // 60

# Compute the current hour
currentHour = totalHours % 24

# Display results
print("Current time is " + str(currentHour) + ":"
    + str(currentMinute) + ":" + str(currentSecond) + " GMT")
```

### 来源：07_数值处理与计算

常用函数：

- time.time(): 返回自Epoch纪元以来的秒数，通常称为Unix时间戳。
- time.localtime(): 返回一个表示本地时间的time.struct_time对象。
- time.gmtime(): 返回一个表示协调世界时（UTC）的time.struct_time对象。

```python
import time

# 获取当前时间戳
timestamp = time.time()
print("当前时间戳:", timestamp)

# 获取本地时间
local_time = time.localtime()
print("本地时间:", local_time)

# 获取UTC时间
utc_time = time.gmtime()
print("UTC时间:", utc_time)
```

**定义：**
- UTC：协调世界时，即世界标准时间。
- Epoch：Unix纪元，即1970年1月1日00:00:00 UTC。
- UNIX时间戳：是从Epoch开始所经过的秒数，不考虑闰秒。

## 1. try ... except ... else ... finally 语句

**来源：03_流程控制**

### 基本概念

- python运行出错会停止并报错——引发异常。
- 通过异常处理机制，可以检查并处理异常，以便使程序不至于突然结束。

### 使用try语句及except子句

- try用于将可能引发异常的代码隔开
- except用于标记当异常出现时需要执行的代码

```python
try:
    num = float(input("Enter a number: "))
except:
    print("Something went wrong!")
```

### 指定异常类型

- 不同错误会引发不同类型的异常
- except子句可以指定类型
- 指定类型是好习惯

```python
try:
    num = float(input("\nEnter a number: "))
except ValueError:
    print("That was not a number!")
```

### 处理多种异常

方法一（合并处理）：

```python
for value in (None, "Hi!"):
    try:
        print("Attempting to convert", value, "-->", end=" ")
        print(float(value))
    except (TypeError, ValueError):
        print("Something went wrong!")
```

方法二（分别处理）：

```python
for value in (None, "Hi!"):
    try:
        print("Attempting to convert", value, "-->", end=" ")
        print(float(value))
    except TypeError:
        print("I can only convert a string or a number!")
    except ValueError:
        print("I can only convert a string of digits!")
```

### 获取异常的参数

```python
try:
    num = float(input("\nEnter a number: "))
except TypeError as e:
    print("I can only convert a string or a number! See: ", e)
except ValueError as e:
    print("That was not a number! Or as Python would say...", e)
except Exception as e:
    print("Something is wrong. As Python would say...", e)
```

### else子句

- 在try语句最后加上else子句代表没有异常时要做的事

```python
try:
    num = float(input("\nEnter a number: "))
except TypeError as e:
    print("I can only convert a string or a number! See: ", e)
except ValueError as e:
    print("That was not a number! Or as Python would say...", e)
except Exception as e:
    print("Something is wrong. As Python would say...", e) 
else:
    print("You entered the number", num)
```

### finally子句

- 在try语句最后加上finally子句代表无论如何要做的事

```python
try:
    num = float(input("\nEnter a number: "))
except TypeError as e:
    print("I can only convert a string or a number! See: ", e)
except ValueError as e:
    print("That was not a number! Or as Python would say...", e)
except Exception as e:
    print("Something is wrong. As Python would say...", e) 
else:
    print("You entered the number", num)
finally:
    print("I'm always here!")
```

## 3. 断言 assert

**来源：03_流程控制**

- 断言：assert 条件(True/False), "条件为False时显示的提示信息"。
- 假如你预期此处x的值为0.3，如果不是0.3，则抛出异常。

```python
>>> x=0.1+0.2
>>> assert x==0.3, "Ooop, x is wrong"
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
AssertionError: Ooop, x is wrong
```

- 断言针对的是程序员的错误，而不是用户的错误。
- 对于可以恢复的错误，请抛出异常，而不是用 assert 语句检测它。
- 运行python时使用 -O 选项，可以禁用断言。
