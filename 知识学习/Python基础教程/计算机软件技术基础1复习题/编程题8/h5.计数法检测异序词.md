# 题目名称：计数法检测异序词

## 题目描述

如果一个字符串只是重排了另一个字符串的字符，那么这个字符串就是另一个字符串的异序词，比如heart和earch，以及python与typhon。为了简化问题，假设要检查的两个字符串长度相同，并且都是由26个英文字母的小写形式组成的。

编写函数 `anagram()`，使用时间复杂度为O(n)的计数法进行检测。

函数头：

​	`def anagram(str1,str2):`

## 输入格式

函数的形参：

​	两个字符串，长度相同，并且都是由26个英文字母的小写形式组成的。

## 输出格式

函数的返回值：

​	一个用尖括号括起来的字符串，内容是两个字符串中出现频率不同的字母，按字母表顺序排列。

## 输入输出样例 #1

### 输入 #1

```
python
typhon
```

### 输出 #1

```
<>
```

## 输入输出样例 #2

### 输入 #2

```
abbcd
abcce
```

### 输出 #2

```
<bcde>
```
