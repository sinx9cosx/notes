# 题目名称：约瑟夫环问题

## 题目描述

有n个人围成一圈，顺序排号（从1开始到n）。从第一个人开始报数（从1开始），凡报到7的倍数（如7，14，21等）或是数字中有7（如7，17，27等）的人退出圈子，报数循环进行直到最后只剩1人为止。问最后留下的人是原来的几号？


## 输入格式

正整数n


## 输出格式

最后留下的人的初始编号


## 输入输出样例

### 输入 #1
```
10
```
### 输出 #1
```
10
```

### 输入 #2
```
5
```
### 输出 #2
```
1
```

### 输入 #3
```
20
```
### 输出 #3
```
5
```
