# 题目名称：找兔子

## 题目描述

山上有n（1<=n<=100）个洞（编号1~n）围成一个圈，有一只狐狸和一个兔子。兔子躲在一个洞中，狐狸希望找到兔子。狐狸先从1号洞找起，第二次隔1个洞找3号洞，第三次隔2个洞找6号洞，以后每次多隔1个洞，依次类推，一共找了1000次也没有找到。请问如果兔子不想被狐狸找到，它可以躲在几号洞中。



## 输入格式

正整数 n 。

## 输出格式

兔子想要不被找到的话，可能躲藏的洞号。多于一种可能的，按洞号从小到大的顺序，一行打印一个洞号。如果不存在可躲藏的洞，打印0。



## 输入输出样例

### 输入 #1
```
3
```
### 输出 #1
```
2
```

### 输入 #2
```
4
```
### 输出 #2
```
0
```

### 输入 #3
```
10
```
### 输出 #3
```
2
4
7
9
```
