# 题目名称：分数的四则运算

## 题目描述

计算二个分数的四则运算结果。

（1）输出结果应该是约分过的最简分数，如10/12应该输出为5/6。

提示：正分数约分为最简分数时，分子分母同时除以它们的最大公约数即可。

（2）如果输出结果的分数为负值，则负号标在分子上，如-3/5，输出-3和5。

（3）如果要输出结果为0时，分母输出1。

（4）如果要结果是个整数时，分母输出1。

（5）输入的分数也遵守上述之规则。

## 输入格式

共输入5个数字，分别是：第1个分数（分别输入分子和分母），运算符（1代表加法，2代表减法，3代表乘法，4代表除法），第2个分数（分别输入分子和分母），每个数字一行。

## 输出格式

计算两个分数的四则运算结果，以分数形式输出，分别输出分子分母，一个一行。

## 输入输出样例

### 输入 #1
```
2
3
2
1
4
```

### 输出 #1
```
5
12
```

### 说明 #1

本例含义为 $ \frac {2} {3} - \frac {1} {4} $，结果输出 $ 5 \over 12$ 。
