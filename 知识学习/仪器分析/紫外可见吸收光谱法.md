# 有机化合物

由**价电子跃迁**产生，主要类型：
- $n →σ^*$
- $π→π^*$
- $n→π^*$
- 电荷转移跃迁

# 无机化合物

- 电荷转移跃迁
- 配位场跃迁

# 常用术语

- 发色团（生色团）：含有$\pi$键的不饱和基团
- 助色团：含孤对电子（杂原子）的饱和基团（使发色团的最大吸收峰向长波方向移动并且增加强度）
- 红移：使吸收峰向长波方向移动
- 蓝移：使吸收峰向短波方向移动
- R带：含杂原子生色团的$n\rightarrow\pi^*$吸收带
- K带：非封闭共轭体系的$\pi\rightarrow\pi^*$跃迁
- B带：芳香族化合物的$\pi\rightarrow\pi^*$

# 影响因素

- 共轭效应：共轭体系增大，$\lambda_{max}$红移，强度增大
- 溶剂效应：溶剂极性增大，$π→π^*$ 跃迁红移，$n→π^*$ 跃迁蓝移
