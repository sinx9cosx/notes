## 题目描述

编写一个函数 `find()`，找到一个二维列表（为一个矩阵，即每行中的元素数量相同，但注意不一定是方阵，即行数不一定等于列数）中的最值以及它们的行列坐标。矩阵的行列坐标均从1开始。

函数头：

​	`def find(matrix, max=True):`

## 输入格式

函数的形参：

​	matrix，一个二维列表。例如，下面是一个三行四列的二维列表：

```
25 89 6 77
89 -32 89 44
2 77 -32 1
```
​	max=True，代表找最大值 ，max=False，代表找最小值，默认为True，找最大值。

## 输出格式

函数的返回值：

​	matrix中的最值及它们的行列坐标。返回值为一个元组，首先是最值，然后是坐标元组。如果最值出现点不唯一，则依次返回所有坐标，一对坐标一组，行坐标较小的先返回，行坐标相同时列坐标较小的先返回。在上面的形参示例中，如果找最大值，值为 89 ，它出现的位置坐标有：`1,2 、2,1 、2,3`  。则函数的返回值为：`(89, ((1, 2), (2, 1), (2, 3)))` 。

## 输入输出样例 #1

### 输入 #1

```
3
25 89 6 77
89 -32 89 44
2 77 -32 1
find(matrix)
```

### 输出 #1

```
89
1,2
2,1
2,3
```
