## 题目描述

编写函数 `isAnagram()`，检查两个单词是否为相似词。

所谓相似词，是指两个单词长度相同，并包含相同的字母（不区分大小写），例如：python 和 typhon 是相似词。

注意如有重复的字母，重复个数也要相同，例如：colourful和fullcolor，虽然长度一样，字母也相同，但是前者有二个u二个l，后者有一个u三个l，故它们不是相似词。

函数头：

​	`def isAnagram(str1, str2):`

## 输入格式

函数的形参：

​	二个字符串 str1, str2

## 输出格式

函数的返回值：

​	布尔值 True/False ，当 str1 与 str2 是相似词时，返回 True，否则返回 False

## 输入输出样例 #1

### 输入 #1

```
isAnagram("python", "typhon")
```

### 输出 #1

```
True
```

## 输入输出样例 #2

### 输入 #2

```
isAnagram("colourful","fullcolor")
```

### 输出 #2

```
False
```
