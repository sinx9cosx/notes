# 题目名称：判断一个点是否在圆内

## 题目描述

已知一个点的坐标（x,y）和一个圆（圆心固定为原点(0,0)，半径r），判断点是否在圆内。比较时，当两个浮点数之差的绝对值小于 0.000001 时可认为二者相等。

两点坐标计算距离的公式为：$ d = \sqrt {(x_1 - x_2)^2 + (y_1-y_2)^2}$

## 输入格式

输入三个数字（整数或小数），代表点的x坐标，y坐标，圆的半径r，一个数字一行。

## 输出格式

若在圆内，输出1；在圆上，输出0；在圆外，输出-1。

## 输入输出样例

### 输入 #1
```
4
5
10
```

### 输出 #1
```
1
```

### 输入 #2

```
9
9
10
```

### 输出 #2

```
-1
```
