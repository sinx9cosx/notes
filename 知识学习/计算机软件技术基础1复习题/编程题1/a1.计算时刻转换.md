# 题目名称：计算时刻转换

## 题目描述

输入一个开始时刻，再输入一个时长（单位：分钟），计算经过这段时间后的结束时刻。

测试输入的数据可以保证两个时刻是同一天，程序不必考虑超过24点的情况，时刻使用 24 小时制，即小时在 0 到 23 之间，分钟在 0 到 59 之间。


## 输入格式

输入3行整数，第一行为开始时刻的小时，第二行为开始时刻的分钟，第三行为从开始时刻到结束时刻之间有多少分钟。

## 输出格式

输出2行，第一行为结束时刻的小时，第二行为结束时刻的分钟。

## 输入输出样例

### 输入 #1
```
9
5
170
```

### 输出 #1
```
11
55
```
