# 题目名称：字母阵列

## 题目描述

输入一个正整数 N，输出 N 行 N 列字母阵列，字母为大写英文字母，第一行从 A 开始，第二行 B 开始，以后每行开头向后错一个字母，当到达 Z，再从 A 开始。每行输出N个字母，字母依次向后，同样也是到达Z时再从A开始。 

## 输入格式

输入一个大于等于 1 的正整数 N。

## 输出格式

输出示例所示的字母阵列。

## 输入输出样例

### 输入 #1
```
5
```

### 输出 #1
```
ABCDE
BCDEF
CDEFG
DEFGH
EFGHI
```

