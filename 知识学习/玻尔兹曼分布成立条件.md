## Q1：玻尔兹曼分布只对与恒温热库接触的系统成立吗？

### 核心回答：不限于恒温热库

**玻尔兹曼分布的本质条件**是——系统处于热力学平衡态、温度 $T$ 有良好定义、且系统/子系统与一个大的"热库"有热接触。这个热库**可以是外部的，也可以是孤立系统自身的"其余部分"**。

### 三种情境分析

| 情境 | 系统类型 | 是否服从玻尔兹曼分布 | 关键说明 |
|:------|:---------|:---------------------|:---------|
| **①** | 正则系综（与外部恒温热库接触） | ✅ 严格服从 | $P_i = e^{-E_i/k_BT} / Q$，这是定义式 |
| **②** | 大孤立系统中的**子系统** | ✅ 渐近服从 | 系统其余部分天然承担"热库"角色，只要 $U \gg \varepsilon$ |
| **③** | 孤立整体系统（微正则系综） | ❌ 不直接适用 | 能量严格固定，所有等能微观态概率相等（等概率公设） |

### 推导核心

从微正则系综出发，考虑大孤立系统中一个粒子的两个能态（$0$ 和 $\varepsilon$），对熵做 Taylor 展开：

$$
S(U - \varepsilon) \approx S(U) - \varepsilon \left.\frac{\partial S}{\partial U}\right|_{V,n} = S(U) - \frac{\varepsilon}{T}
$$

代入微观状态数之比：

$$
\frac{P_\varepsilon}{P_0} = \frac{w(U-\varepsilon)}{w(U)} = \frac{e^{S(U-\varepsilon)/k}}{e^{S(U)/k}} = e^{-\varepsilon/kT}
$$

> [!quote] 关键洞察
> "温度是一个衍生量，而非体系微观的状态物理量。"——玻尔兹曼分布自然浮现，**无论 $T$ 由外部热库维持还是由孤立系统内部自由度统计平均产生**。

### 物理图景：$k_BT$ 作为能量标杆

- 若两态能量差为 $k_BT$，分布比为 $1 : 2.72 \ (\approx 1:e)$
- **高于** $k_BT$ 的态 → 难以占据
- **低于** $k_BT$ 的态 → 易占据

## Q2：什么是"良好定义"的温度 $T$？

### 核心定义

$$
\boxed{T \equiv \left(\frac{\partial U}{\partial S}\right)_{V,n}}
$$

- 下标 $V$：保证变化过程只有传热而无体积功
- 下标 $n$：保证粒子数不变

### "良好定义"的四个条件

| # | 条件 | 具体要求 |
|:--|:-----|:---------|
| **①** | **热力学平衡态** | 系统内部达成热平衡，$S(U)$ 函数本身有意义 |
| **②** | **$C_v > 0$（热力学稳定性）** | $S(U)$ 曲线严格上凸（$\partial^2 S/\partial U^2 < 0$），保证系统自发趋于温度均衡而非放大温差 |
| **③** | **热力学极限** | 系统足够大，统计涨落可忽略，$S(U)$ 偏导数光滑 |
| **④** | **$S(U)$ 单调递增（通常情况）** | 保证 $T > 0$；特殊情况（布居数反转）可出现负温度 |

### 温度的三重物理含义

1. **熵变化难易标度**：$T$ 衡量传递单位能量引起的熵变大小——温度越低，同能量传递导致的熵增越大
2. **冷热程度标度**：$T$ 数值对应日常经验冷热
3. **传热方向判据**：热自发从高温流向低温——因为低温系统"用相同能量能产生更多熵增"

> [!important] 传热方向的本质
> 热传递是全部系统**最大化总熵**的自发能量传递过程。相同能量转移下，熵增更多的体系是温度较低的体系。

### 温度的平衡判据

$T_1 = T_2$ 是热平衡的**充要条件**：

- 已知 $\left(\frac{\partial U_1}{\partial S_1}\right) = \left(\frac{\partial U_2}{\partial S_2}\right)$，且 $dU_1 = -dU_2$
- 则 $dS_1 = -dS_2$，$dS = dS_1 + dS_2 = 0$
- 系统总熵达到极值（最大值）→ **热平衡**

### 延伸：负温度

当高能态粒子数超过低能态（**布居数反转**），$S$ 随 $U$ 增加而**减小** → $\partial U/\partial S < 0$ → $T < 0$。

- 负温度系统比**任何**正温度系统"更热"
- 热量从负温度系统流向正温度系统

```
│                                                          │

│    ├─ 等概率公设：所有可及微观态概率相等                     │
│    ├─ 温度 T = (∂U/∂S)_{V,N} 由熵函数导出                 │
│    └─ 子系统 → 自然呈现玻尔兹曼分布                         │
│         ↓                                                │
│  正则系综 (T, V, N 固定)                                  │
│    ├─ 玻尔兹曼分布：P_i ∝ e^{-E_i/k_BT}                  │
│    ├─ 配分函数 Q = Σ e^{-E_i/k_BT} 联系微观与宏观          │
│    └─ 内能 U = N k_B T² (∂ln q/∂T)                       │
│         ↓                                                │

│    └─ 粒子数也涨落，需引入化学势 μ                          │
│                                                          │
│  核心纽带：温度 T 在所有系综中都是关键参数                   │
│  根源：T ≡ (∂U/∂S)_{V,n}，是熵函数的衍生量                 │
└──────────────────────────────────────────────────────────┘
```

## 关联概念

- **[[配分函数]]**：$q = \sum_i g_i e^{-\varepsilon_i/k_BT}$，联系微观状态分布与宏观热力学性质
- **[[等概率公设]]**：所有满足系统边界条件的微观状态出现概率相等——统计力学的逻辑出发点
- **[[最概然分布]]**：玻尔兹曼分布是微观状态数 $W$ 取极大值时的分布，可代表系统的全部状况
- **[[熵的统计诠释]]**：$S = k \ln W$（Boltzmann 熵公式）

## 对比辨析

| 对比维度 | 微正则系综 | 正则系综 |
|:---------|:-----------|:---------|
| 固定量 | $U, V, N$ | $T, V, N$ |
| 能量 | 严格固定，无涨落 | 有涨落，$\langle E \rangle = U$ |
| 分布 | 等概率（等能面上） | $P_i \propto e^{-E_i/k_BT}$ |
| 温度来源 | $T = (\partial U/\partial S)_{V,N}$ | $T$ 由外部热库给定 |
| 等价性 | $N \to \infty$ 时与正则系综等价 | — |

## 自测问题

1. **小系统极限**：若孤立系统只有 10 个粒子，总能量固定为 $3\varepsilon$（每粒子只有 0 和 $\varepsilon$ 两能级），此时单粒子分布是否严格服从玻尔兹曼分布？$U \gg \varepsilon$ 条件对 $N$ 有何隐含要求？

2. **涨落与温度**：正则系综中系统总能量会涨落，为什么我们仍能定义一个"恒定"的温度 $T$？系统自身的瞬时温度也在涨落吗？

3. **负温度逻辑**：当负温度系统与正温度系统接触，能量为何从"负"流向"正"？这与"热从高温流向低温"的这一常识是否矛盾？请用 $S-U$ 曲线解释。

4. **系综等价性**：解释为什么在热力学极限 $N \to \infty$ 下，微正则系综和正则系综给出相同的热力学性质。这暗示玻尔兹曼分布具有怎样的"普适性"？
