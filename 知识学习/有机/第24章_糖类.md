## 24.1 糖类简介

- **定义**：糖类（碳水化合物）是多羟基醛或多羟基酮。最初被误认为是碳的水合物 C_n(H_2O)_n，现已修正，但"碳水化合物"名称沿用至今。
- **自然来源**：植物通过**光合作用**将 CO_2 和 H_2O 转化为葡萄糖。葡萄糖中的 C-C 和 C-H 键能量高于 CO_2 的 C=O 键，人体通过氧化葡萄糖释放能量。
- **功能**：能量来源、结构支撑（木材、龙虾壳等）；DNA 也由糖类衍生物组成。
- **代表化合物**：葡萄糖（glucose，己醛糖）、果糖（fructose，己酮糖）。

## 24.2 单糖的分类

### 醛糖与酮糖

| 分类依据 | 说明 |
|----------|------|
| 醛糖 (aldose) | 含醛羰基（如葡萄糖） |
| 酮糖 (ketose) | 含酮羰基（如果糖） |

按碳原子数进一步分类，如 **戊醛糖**（5C醛糖）、**己酮糖**（6C酮糖）。三个命名要素：醛/酮 + 碳数（丙/丁/戊/己/庚）+ "糖"。

### D和L糖

- 以**(+)-甘油醛**（D-甘油醛）为参照标准。甘油醛只有一个手性中心。
- **Fischer-Rosanoff规则**：单糖在Fischer投影式中，离羰基最远的手性中心上 OH 在右侧为 **D-糖**，在左侧为 **L-糖**。
- 自然界绝大多数糖为 **D-糖**。D-甘油醛右旋，但其他D-糖不一定右旋（如 D-赤藓糖和 D-苏阿糖均为左旋）。
- "D"指离羰基最远手性中心的构型，不指旋光方向。

## 24.3 醛糖的构型

醛糖族谱从 **D-甘油醛** 开始，在羰基下方逐级插入新手性中心：

| 类别 | 手性中心数 | 立体异构体数 | 代表D-糖 |
|------|-----------|-------------|----------|
| 丁醛糖 (tetrose) | 2 | 4 (2对对映体) | D-赤藓糖, D-苏阿糖 |
| 戊醛糖 (pentose) | 3 | 8 (4对对映体) | D-核糖(RNA组分), D-阿拉伯糖, D-木糖, D-来苏糖 |
| 己醛糖 (hexose) | 4 | 16 (8对对映体) | D-阿洛糖, D-阿卓糖, **D-葡萄糖**, D-甘露糖, D-古洛糖, D-艾杜糖, D-半乳糖, D-塔洛糖 |

**D-葡萄糖**是最常见和最重要的糖，是本章后续讨论的核心。

---

## 24.4 酮糖的构型

酮糖族谱从**二羟基丙酮**开始构建。由于羰基在C2位，比同碳数醛糖少一个手性中心。

- **D-己酮糖**仅有4种（vs. 醛糖的8种）：D-阿洛酮糖、**D-果糖**、D-山梨糖、D-塔格糖。
- **D-果糖**是自然界最常见的酮糖。D-果糖和D-葡萄糖互为构造异构体。

## 24.5 单糖的环状结构

### 吡喃糖式 (Pyranose)

- 羟基醛在酸催化下发生**分子内环合**，形成环状半缩醛。六元环以"吡喃"命名。
- D-葡萄糖 → D-吡喃葡萄糖（几乎100%为环状，开链式 <0.01%）。
- **端基碳（C1）**：环合后成为新手性中心，产生两种**端基异构体 (anomers)**：
  - **alpha-端基异构体**：C1-OH 与 C5-CH2OH 处于**反式**（trans，OH朝下）
  - **beta-端基异构体**：C1-OH 与 C5-CH2OH 处于**顺式**（cis，OH朝上）
- 平衡组成：beta 63%，alpha 37%。

### 变旋现象 (Mutarotation)

- alpha 和 beta 端基异构体通过开链式相互转化。
- 纯 alpha：[alpha] = +112.2°；纯 beta：[alpha] = +18.7°；平衡混合物：[alpha] = +52.6°。
- 酸或碱催化加速变旋。

### Haworth投影式规则

- Fischer式中**右侧OH → Haworth中朝下**
- Fischer式中**左侧OH → Haworth中朝上**
- 氧原子置于环的**右后方**
- D-糖：C5-CH2OH 朝上

### 椅型构象

- D-葡萄糖的所有大取代基（OH 和 CH2OH）均在**平伏键**上 → 特别稳定 → 解释了D-葡萄糖为何是最常见的天然单糖。
- 椅型构象绘制规则：氧原子在右后上方，先画Haworth，转换为椅型，最大取代基优先放在平伏键。

### 呋喃糖式 (Furanose)

- 五元环状半缩醛，以"呋喃"命名。
- D-果糖：开链式 + 两种吡喃糖 + 两种呋喃糖的平衡（beta-吡喃糖70%为主要形式，但生物体内主要参与代谢的是beta-呋喃糖式）。

## 24.6 单糖的反应

### 1. 酯和醚的生成

| 反应类型 | 条件 | 产物 |
|----------|------|------|
| **酯化** | 过量 Ac2O / AcCl，吡啶 | 五-O-乙酰基-beta-D-吡喃葡萄糖（全酯化） |
| **醚化** | 过量 CH3I，Ag2O（弱碱） | 五-O-甲基醚（Williamson醚合成变体） |

> 不能用强碱，防止副反应（如差向异构化）。

### 2. 糖苷的生成 (Glycoside Formation)

- 半缩醛 + 醇 → 缩醛（**糖苷**）。机理：
  1. 端基OH质子化、脱水 → 碳正离子（因邻位O共振稳定）
  2. 醇进攻碳正离子 → 脱质子 → 糖苷
- 命名：烷基在前 + "苷"（如**甲基-alpha-D-吡喃葡萄糖苷**）
- 糖苷在中性/碱性条件下稳定，不会变旋；酸性条件下水解回半缩醛。
- 产物分布与起始端基异构体类型无关（均经平面形碳正离子中间体）。

### 3. 差向异构化 (Epimerization)

- D-葡萄糖在强碱（NaOH/H2O）中经**烯二醇中间体** → D-葡萄糖 + D-甘露糖的混合物。
- D-葡萄糖和D-甘露糖互为**C2-差向异构体**（仅C2构型不同）。
- 化学家避免将糖暴露于强碱性条件。

### 4. 还原 (Reduction)

- NaBH4 还原醛糖/酮糖 → **糖醇**（如 D-葡萄糖 → D-山梨糖醇）。
- 开链式被还原 ← 平衡移动 → 直至几乎全部转化。
- D-山梨糖醇存在于水果浆果中，常用作食品中糖的替代品。

### 5. 氧化 (Oxidation)

| 氧化剂 | 适用范围 | 产物 |
|--------|----------|------|
| Br2/H2O, pH=6 | 仅**醛糖**（酮糖不反应） | 糖酸（如 D-葡萄糖酸） |
| Tollens试剂 (Ag+/NH3) | 醛糖 + 酮糖 | Ag镜 |
| Fehling试剂 (Cu2+/酒石酸钠) | 醛糖 + 酮糖 | Cu2O 红色沉淀 |
| Benedict试剂 (Cu2+/柠檬酸钠) | 醛糖 + 酮糖 | Cu2O 红色沉淀 |
| HNO3（强） | 醛糖 | 糖二酸（醛基+伯醇均被氧化） |

- **还原糖**：对Tollens/Fehling/Benedict测试呈阳性的糖（含游离半缩醛羟基）。
- **糖苷不是还原糖**（端基碳被保护为缩醛，无法开环）。
- Br2/H2O 体系可区分醛糖（反应）和酮糖（不反应）。
- Tollens/Fehling/Benedict 可区分还原糖和糖苷。

### 6. 碳链增长：Kiliani-Fischer合成

- **总转化**：醛糖 → 碳链 +1 个碳的醛糖（如戊醛糖 → 己醛糖）
- **步骤**：
  1. HCN 与醛羰基亲核加成 → 氰醇（非立体选择性，得C2差向异构体混合物）
  2. 氰基还原/水解 → 醛基
- **现代改进版**：Pd/BaSO4（Lindlar催化剂的中毒变体）催化氢化，氰基 → 亚胺 → 水解得醛，一步完成。
- 应用实例：D-阿拉伯糖 → D-葡萄糖 + D-甘露糖（C2差向异构体）。

## 重要机理汇总

1. **酸催化环合**：OH 进攻质子化羰基 → 环状半缩醛（机理20.5延伸）
2. **变旋**：开链式 ↔ alpha-吡喃糖 ↔ beta-吡喃糖
3. **糖苷形成**：端基OH质子化脱水 → 共振稳定碳正离子 → 醇进攻 → 糖苷
4. **差向异构化**：碱催化烯二醇互变异构 → C2构型翻转
5. **Kiliani-Fischer**：HCN亲核加成 → 氰醇 → 还原/水解为醛

---

## 实用要点

- **区分醛糖和酮糖**：Br2/H2O (pH=6) 仅氧化醛糖。
- **鉴别还原糖**：Tollens/Fehling/Benedict 测试。还原糖 = alpha/beta 端基异构体（不是糖苷）。
- **保护糖类**：酯化或醚化使其溶于有机溶剂便于纯化；糖苷化保护端基。
- **避免强碱**：防止差向异构化导致产物复杂化。
