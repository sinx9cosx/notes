# 氨基酸

- **氨基酸是蛋白质的基本结构单元**，其中α-氨基酸的氨基和羧基连接在同一个碳原子上，是自然界中最常见的形式。  
  解释：α-氨基酸的氨基连在羧基的α碳上，这个碳通常是手性中心（甘氨酸除外）。

- **蛋白质中常见的20种氨基酸根据侧链性质分为非极性、极性和带电侧链**，除甘氨酸外均为L-构型。  
  解释：侧链决定氨基酸的种类和化学性质，天然蛋白质几乎全由L-氨基酸组成。

- **氨基酸在生理pH下以两性离子形式存在**，羧基去质子化、氨基质子化，使其既可作为酸也可作为碱。  
  解释：这种内盐结构使氨基酸易溶于水、熔点高，并具有缓冲能力。

- **每种氨基酸有特定的等电点（pI）**，在该pH下两性离子浓度最高，分子整体净电荷为零。  
  解释：pI由氨基酸的pKa值决定，可用于电泳分离不同氨基酸。

- **实验室合成氨基酸常用α-卤代羧酸氨解、氨基丙二酸酯合成和Strecker反应**，这些方法通常得到外消旋混合物。  
  解释：这些经典方法操作简单，适合制备多种α-氨基酸，但缺乏立体选择性。

- **通过不对称氢化等手性催化技术可合成高对映体纯度的L-或D-氨基酸**，如L-多巴的工业生产。  
  解释：手性催化剂控制反应立体化学，避免低效率的拆分过程。

# 肽

- **氨基酸通过肽键（酰胺键）连接形成肽和蛋白质**，短链为肽，长链（>40-50个残基）为蛋白质。  
  解释：肽键具有部分双键性质，导致旋转受限，影响肽的空间结构。

- **肽链有方向性：N端（自由氨基）和C端（自由羧基）**，书写时N端在左。  
  解释：序列从N端到C端书写，是描述肽和蛋白质一级结构的标准方式。

- **半胱氨酸残基可通过氧化形成二硫键**，稳定肽链的三维结构。  
  解释：二硫键在蛋白质折叠和稳定性中起关键作用，常见于分泌蛋白和激素。

- **Edman降解用于从N端逐一鉴定氨基酸序列**，适用于少于50个残基的肽。  
  解释：异硫氰酸苯酯与N端氨基酸反应，生成PTH衍生物并释放该氨基酸，可自动化分析。

- **大蛋白需用胰蛋白酶或糜蛋白酶等酶切为小片段后再测序**，通过重叠片段推断完整序列。  
  解释：胰蛋白酶切在Lys/Arg的C端，糜蛋白酶切在Phe/Tyr/Trp的C端，提供特定断裂点。

- **肽合成需保护氨基和羧基以避免副反应**，常用Boc或Fmoc保护氨基，酯保护羧基。  
  解释：DCC等偶联试剂促进肽键形成，保护基策略实现区域选择性合成。

- **Merrifield固相合成法将肽链连在不溶性树脂上**，可自动化合成数十个氨基酸长的肽。  
  解释：通过反复偶联、洗涤、脱保护，高效合成多肽，如核糖核酸酶。

# 蛋白质

- **蛋白质结构分为四个层次**：一级结构（氨基酸序列）、二级结构（α螺旋和β折叠）、三级结构（三维折叠）、四级结构（亚基组装）。  
  解释：一级结构决定高级结构；二级结构由氢键稳定；三级结构由侧链相互作用形成；四级结构由多条肽链组成。

- **α螺旋中每个羰基与第四个残基的氨基形成氢键**，侧链向外伸展；β折叠通过链间氢键稳定。  
  解释：脯氨酸因无法提供NH氢键且构象刚性，常破坏α螺旋。

- **蛋白质变性是二级和三级结构被破坏但一级结构保留的过程**，如煮鸡蛋使蛋白凝固。  
  解释：加热、极端pH或有机溶剂可导致变性，通常不可逆并伴随功能丧失。

- **蛋白质功能多样**：结构蛋白（如角蛋白）、酶、转运蛋白（如血红蛋白）、激素等。  
  解释：功能高度依赖三维结构，一个氨基酸突变（如镰刀型贫血）可导致严重疾病。

- **朊病毒是错误折叠的蛋白质**，可引发致命神经退行性疾病，如疯牛病和克-雅病。  
  解释：朊病毒诱导正常蛋白质同样错误折叠并聚集，是结构与功能关系的极端实例。
