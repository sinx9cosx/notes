## 27.1 合成聚合物简介

### 基本概念

- **聚合物**是由**单体**通过共价键重复连接而成的大分子。
- 聚合物的重复单元外加方括号，下标 *n* 表示大量重复单元聚合度。
- 聚合物分两大类：
  - **合成聚合物**：由科学家在实验室或工厂中制备。
  - **生物聚合物**：由生命体产生（如 [[多糖]]、[[DNA]]、[[RNA]]、[[蛋白质]]，见前面章节）。

### 重要性

- 过去一个世纪，聚合物已渗透到社会生活各方面：咖啡杯、软饮料瓶、合成织物、CD/DVD、垃圾袋、人造心脏瓣膜、汽车部件等。
- 仅美国每年生产超过50万亿磅合成聚合物。
- 本章专门讨论**合成聚合物**，生物聚合物已在 [[第24章]]、[[第25章]] 讨论。

## 27.2 合成聚合物的命名

### 命名方式

1. **IUPAC系统命名**：存在但很少使用。
2. **按单体命名**（IUPAC认可，最通用）：在单体名称前加"聚"字。
   - 丙烯 → 聚丙烯 (polypropylene)
   - 苯乙烯 → 聚苯乙烯 (polystyrene)
3. **商品名**：特氟龙 (Teflon，聚四氟乙烯)、凯芙拉 (Kevlar) 等。

### 常见聚合物及用途（表27.2）

| 聚合物名称 | 单体 | 主要应用 |
|---|---|---|
| **聚乙烯** (PE) | 乙烯 CH2=CH2 | 瓶子、垃圾袋 |
| **聚丙烯** (PP) | 丙烯 CH2=CHCH3 | 地毯纤维、电器、汽车轮胎 |
| **聚异丁烯** | 异丁烯 CH2=C(CH3)2 | 填缝密封胶、自行车内胎、篮球 |
| **聚苯乙烯** (PS) | 苯乙烯 CH2=CHPh | 绝缘泡沫、电视、收音机 |
| **聚氯乙烯** (PVC) | 氯乙烯 CH2=CHCl | 水管、乙烯基塑料 |
| **聚α-氰基丙烯酸甲酯** | α-氰基丙烯酸甲酯 | 强力胶（瞬干胶） |
| **聚四氟乙烯** (PTFE) | 四氟乙烯 CF2=CF2 | 不粘涂层（特氟龙） |

## 27.3 共聚物

### 定义

- **均聚物** (homopolymer)：由一种单体构成的聚合物。
- **共聚物** (copolymer)：由两种或两种以上不同类型的单体构成的聚合物。

### 共聚物类型

按单体结合顺序：
- **交替共聚物** (alternating copolymer)：单体交替排列。
- **无规共聚物** (random copolymer)：单体随机分布。
- **嵌段共聚物** (block copolymer)：同类单体聚集成段。
- **接枝共聚物** (graft copolymer)：主链上接有不同单体的侧链。

### 常见共聚物及用途（表27.3）

| 共聚物名称 | 单体 | 应用 |
|---|---|---|
| **莎纶** (Saran) | 氯乙烯 + 偏二氯乙烯 | 食品包装 |
| **苯乙烯-丙烯腈共聚物** (SAN) | 苯乙烯 + 丙烯腈 | 洗碗机安全厨具、电池盒 |

共聚物的性质通常与相应的均聚物不同，通过调节共聚单体的比例可以调控材料性能。

## 27.4 聚合物分类：按反应类型

### 加成聚合物 (Addition Polymers)

- 单体通过**加成反应**聚合，没有小分子副产物生成。
- 单体通常含有 C=C 双键（烯烃、乙烯基单体）。
- 机理包括自由基聚合、阳离子聚合、阴离子聚合。
- 典型实例：聚乙烯、聚丙烯、聚苯乙烯、聚氯乙烯、聚四氟乙烯。

### 缩合聚合物 (Condensation Polymers)

- 聚合过程中伴随**小分子**（如 H2O、HCl、CH3OH）的消去。
- 单体通常含有两个官能团。
- 典型实例：
  - **聚酯**：二酸 + 二醇 → 聚酯 + 水（如 PET）
  - **聚酰胺**：二酸 + 二胺 → 聚酰胺 + 水（如尼龙）
  - **聚碳酸酯**、**聚氨酯**等。

## 27.5 聚合物分类：按组装方式

### 链增长聚合 (Chain-Growth Polymerization)

- 单体逐个添加到增长链的活性末端。
- 需要**引发剂**（自由基引发剂、酸、碱等）。
- 包括自由基聚合、阳离子聚合、阴离子聚合。
- 链增长快，分子量迅速增大。

### 逐步增长聚合 (Step-Growth Polymerization)

- 任何两个含有互补官能团的分子都可以相互反应。
- 单体先形成二聚体、三聚体等低聚物，然后逐步连接。
- 缩合聚合多属于逐步增长。
- 分子量缓慢增长，需要高转化率才能获得高分子量。

## 27.6 聚合物分类：按结构

- **线形聚合物** (linear)：单链结构，可溶解、可熔融，如高密度聚乙烯 (HDPE)。
- **支化聚合物** (branched)：主链上带有侧链分支，如低密度聚乙烯 (LDPE)。
- **交联聚合物** (cross-linked)：链间通过共价键连接成三维网络，不溶不熔。如硫化橡胶、环氧树脂。
- **网络聚合物** (network)：高度交联的结构。

---

## 27.7 聚合物分类：按性质

| 类型 | 特征 | 实例 |
|---|---|---|
| **热塑性** (thermoplastic) | 加热软化/熔融、冷却硬化，可逆，可回收 | PE、PP、PS、PVC、PET |
| **热固性** (thermoset) | 加热后永久硬化、不可逆，不可回收 | 环氧树脂、酚醛树脂、硫化橡胶 |
| **弹性体** (elastomer) | 高弹性，可大幅拉伸并恢复原状 | 天然橡胶、合成橡胶（丁苯橡胶、氯丁橡胶） |

## 27.8 聚合物回收

- 聚合物废弃物是环境的重要挑战。
- **回收策略**：
  - **机械回收**：直接再加工（适用于热塑性聚合物）。
  - **化学回收**：将聚合物解聚为单体再重新聚合。
  - **能量回收**：焚烧回收热能。
- 塑料分类编码（树脂识别码 1-7）：
  - 1=PET、2=HDPE、3=PVC、4=LDPE、5=PP、6=PS、7=其他。
- 热固性聚合物和交联聚合物难以回收。

## 重要专业术语

| 术语 | 英文 | 含义 |
|---|---|---|
| 单体 | monomer | 构成聚合物的重复单元原料 |
| 聚合物 | polymer | 由单体重复连接而成的大分子 |
| 均聚物 | homopolymer | 由一种单体构成的聚合物 |
| 共聚物 | copolymer | 由两种及以上单体构成的聚合物 |
| 加成聚合 | addition polymerization | 不产生小分子副产物的聚合 |
| 缩合聚合 | condensation polymerization | 伴随小分子消去的聚合 |
| 链增长 | chain-growth | 单体逐个加到活性链末端 |
| 逐步增长 | step-growth | 任意两分子均可反应的聚合模式 |
| 热塑性 | thermoplastic | 可反复加热塑形的聚合物 |
| 热固性 | thermoset | 加热后不可逆硬化的聚合物 |
| 弹性体 | elastomer | 具有高弹性的聚合物 |
| 交联 | cross-linking | 聚合物链间的共价键连接 |
| 聚合度 | degree of polymerization | 聚合物中重复单元数 *n* |
