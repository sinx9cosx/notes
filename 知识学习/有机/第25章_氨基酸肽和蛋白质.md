## 25.1 氨基酸、肽和蛋白质简介

### 基本概念
- **蛋白质**是由**氨基酸**单体通过酰胺键（肽键）连接而成的聚合物。
- **α-氨基酸**：氨基和羧基连接在同一个碳原子（α-碳）上，仅间隔一个碳原子。
- α-碳是**手性中心**（R ≠ H时），天然氨基酸几乎全是L-构型（甘氨酸除外，R=H，非手性）。

### 肽与蛋白质的区分
- **肽键**：氨基酸之间形成的酰胺键（—CO—NH—）。
- **二肽**：两个氨基酸连接；**三肽**：三个；**多肽**：通常<40~50个残基。
- **蛋白质**：更长的多肽链（>50个残基）。
- **酶**：催化活细胞中大多数反应的蛋白质，人体需要约50000种以上的酶。

### 立体化学
- 天然蛋白质中的氨基酸几乎均为**L-构型**（Fischer投影式中氨基在左侧，类比L-甘油醛）。
- 少数D-氨基酸存在于细菌的蛋白质和多肽抗生素中（如杆菌肽、多黏菌素）。
- 除半胱氨酸为R构型外，所有天然L-氨基酸的α-碳均为**S构型**（Cahn-Ingold-Prelog规则）。

### 其他天然氨基酸（非蛋白质组成）
- **GABA**（γ-氨基丁酸）：神经递质，存在于大脑中。
- **甲状腺素**（Thyroxine）：甲状腺激素。

### 必需氨基酸（10种）
人体不能足量合成，须从饮食获取：异亮氨酸、亮氨酸、甲硫氨酸、苯丙氨酸、苏氨酸、色氨酸、缬氨酸、**精氨酸、组氨酸、赖氨酸**。
- **完全蛋白质**：含所有10种必需氨基酸（肉、鱼、牛奶、鸡蛋）。
- **不完全蛋白质**：缺少一种或多种（大米缺赖氨酸和苏氨酸；玉米缺赖氨酸和色氨酸；豆类缺甲硫氨酸）。

### 酸碱性
- 氨基酸在水溶液中以**两性离子**（Zwitterion）形式存在：—NH3+ 和 —COO-。
- **pKa1**（α-COOH）：约2~3，生理pH下去质子化为—COO-。
- **pKa2**（α-NH3+）：约9~10，生理pH下保持质子化为—NH3+。
- **pKa3**：仅存在于有酸性或碱性侧链的氨基酸。
- 氨基酸具有**两性**：既可作酸（与碱反应，铵离子去质子），又可作碱（与酸反应，羧酸根质子化）。

### 等电点（pI）
- 两性离子浓度最大时的pH值。
- 无酸/碱性侧链：pI = (pKa1 + pKa2) / 2。例：丙氨酸 pI = (2.34 + 9.69) / 2 = 6.02。
- 酸性侧链氨基酸（Asp, Glu）：pI = 两个羧基pKa的平均值。
- 碱性侧链氨基酸（Lys, Arg, His）：pI = 两个氨基pKa的平均值。

### 电泳分离
- 基于pI差异分离氨基酸混合物。
- pH < pI：氨基酸带正电，向阴极移动。
- pH > pI：氨基酸带负电，向阳极移动。
- 检测方法：**茚三酮**（Ninhydrin）溶液处理→加热→紫色斑点。紫色斑点数目=不同氨基酸种类数。

### 茚三酮反应
- 氨基酸 + 茚三酮 → 紫色产物 + CO2 + H2O + 醛（RCHO）。
- 氨基酸的N原子进入紫色产物，其余部分降解。
- 脯氨酸（二级胺）与茚三酮产生黄色而非紫色。
- **法医应用**：用于显现隐藏指纹（指纹汗液中的氨基酸与茚三酮反应产生荧光紫色图像）。

## 25.3 氨基酸的合成

### 方法一：由α-卤代羧酸合成（Hell-Volhard-Zelinsky反应）
1. HVZ反应：羧酸 + Br2/PBr3 → α-溴代羧酸。
2. SN2反应：α-溴代羧酸 + 过量NH3 → α-氨基酸（外消旋）。
- 多烷基化不是问题，因为底物空间位阻大。

### 方法二：氨基丙二酸酯合成（Amidomalonate Synthesis）
- 原料：**乙酰氨基丙二酸二乙酯**（Diethyl acetamidomalonate）。
- 步骤：
  1. NaOEt夺氢，形成烯醇负离子。
  2. 与卤代烷（RX）进行SN2烷基化。
  3. H3O+/加热：水解+脱羧，同时酰胺水解→α-氨基酸（外消旋）。
- 限制：RX必须适合SN2反应（不可用叔卤代烷）。
- 甘氨酸无需卤代烷（直接水解脱羧即可）。

### 方法三：Strecker合成
- 两步反应：
  1. 醛 + NH4Cl + NaCN → α-氨基腈（机理25.1：亚胺形成→氰基亲核进攻）。
  2. α-氨基腈 + H3O+/加热 → 腈水解为羧基→α-氨基酸（外消旋）。
- 产物种类由初始醛决定。

### L-氨基酸的对映选择性合成
- 上述三种方法均得到**外消旋混合物**，需拆分或不对称合成获得光学纯产物。
- **Knowles不对称氢化**：使用手性催化剂（如Ru-BINAP配合物）对烯烃进行对映选择性氢化。
- 实例：L-多巴（L-dopa）的合成，ee值可高达99%。
- 甘氨酸非手性，无需手性催化剂。

## 25.4 肽的结构

### 肽的组装
- 氨基酸通过**肽键**（酰胺键）连接，脱水缩合形成。
- 实例：**谷胱甘肽**（Glutathione, Glu-Cys-Gly）：三肽，含两个肽键，作为自由基和亲电试剂的清除剂存在于所有哺乳动物中。
- 连接顺序重要：Ala-Gly 与 Gly-Ala 是**构造异构体**。

### N端与C端
- **N端**：含自由氨基（—NH2），写在左侧。
- **C端**：含自由羧基（—COOH），写在右侧。
- 肽序列按N→C方向用三字母或单字母缩写表示。

### 肽键的几何结构
- 酰胺键有重要共振极限结构，C—N键具有**部分双键性质**。
- 氮原子为**sp²杂化**，呈平面构型。
- C—N键旋转受限，产生两种构象：
  - **s-反式**（s-trans）：更稳定，无空间张力。
  - **s-顺式**（s-cis）：存在空间位阻，能量较高。
- 多肽中每个肽键倾向于取s-反式构象，但其他σ键可自由旋转，因此多肽可采取多种构象。

### 二硫键
- 两个半胱氨酸残基的—SH氧化偶联形成—S—S—。
- 可在**同一条链内**（链内二硫键）或**不同链间**（链间二硫键）形成。
- 对肽和蛋白质的三维结构有重大影响。

### 生物学重要的肽
| 肽名 | 结构特征 | 功能 |
|------|----------|------|
| 亮氨酸脑啡肽 Leu-enkephalin | 五肽 | 脑中控制疼痛，与吗啡受体作用 |
| 甲硫氨酸脑啡肽 Met-enkephalin | 五肽（仅最后一个残基与上不同） | 同上 |
| 缓激肽 Bradykinin | 九肽 | 扩张血管、消炎 |
| 加压素 Vasopressin | 含二硫键，C端酰胺化 | 肾脏水再吸收、控制血压 |
| 催产素 Oxytocin | 含二硫键，与加压素仅两残基不同 | 促进分娩和泌乳 |
| 阿斯巴甜 Aspartame | Asp-Phe-OCH3（二肽甲酯） | 人工甜味剂（NutraSweet） |
| 杆菌肽A Bacitracin A | 环状多肽，含D-氨基酸 | 多肽抗生素，抑制细菌细胞壁合成 |
| 多黏菌素B Polymyxin B | 环状多肽，含D-氨基酸 | 多肽抗生素 |

> 加压素和催产素仅有两个残基不同，但功能截然不同——说明**结构微小差异可导致功能巨大差异**。

## 25.5 肽测序

### Edman降解（Edman Degradation）
- 从N端逐一去除氨基酸残基并鉴定。
- 试剂：**异硫氰酸苯酯**（Ph-N=C=S）+ **三氟乙酸**（CF3COOH）。
- N端残基转化为**苯基乙内酰硫脲**（PTH）衍生物，通过分析PTH衍生物确定氨基酸种类。
- 过程可自动化（自动测序仪），一次最多测约50个残基。
- 机理25.2：亲核进攻→分子内环化→重排→PTH衍生物。

### 酶切法
- 大肽（>50残基）先酶切成小片段，再分别测序。
- **胰蛋白酶**（Trypsin）：水解**精氨酸和赖氨酸**的羧基端肽键（碱性氨基酸C端）。
- **糜蛋白酶**（Chymotrypsin）：水解**苯丙氨酸、酪氨酸和色氨酸**的羧基端肽键（芳香族氨基酸C端）。
- 用不同酶切得到的片段交叉对比，可推断完整序列。
