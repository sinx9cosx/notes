# 摘要

**摘要：** 苯醌是一种简单、常见的具有氧化还原活性的有机分子。苯醌及其衍生物在生物体内的电荷转移过程中扮演着重要的中间体的角色，其氧化还原特性与其衍生物的分子结构相关。本实验结合循环伏安和量子化学计算的方法研究苯醌、萘醌和四甲基苯醌（杜醌）的氧化还原电势，探讨其氧化还原电势的分子结构基础。

本实验立足本科学习所涉及的基础物理化学原理和有机分子的性质，综合运用实验与理论方法，探索宏观电化学性质与分子结构间的关系。通过实验与理论分析，探索微观分子结构与宏观性质之间的逻辑关系，更深入理解量子化学、有机化学、统计力学及热力学的基本理论，实现跨越尺度的贯通式的学习。

# 实验目的

1. 学习循环伏安法的原理和实验技术，利用循环伏安法测量苯醌及其衍生物的标准氧化还原电势。
2. 学习量子化学计算的基本原理及量子化学计算软件 Gaussian 的使用方法；利用 Gaussian 计算苯醌及其衍生物的电子结构和热力学性质。
3. 通过 Gaussian 计算得到的热力学数据，计算苯醌及其衍生物的氧化还原电势；与循环伏安方法相对照，验证理论与实验的符合程度，考察苯醌及其衍生物氧化还原电势差别的来源。
4. 通过 Gaussian 计算得到的结果，研究其分子轨道能量与氧化还原电势的关系，考察氧化还原电势变化的分子结构基础。

# 前言

苯醌是一种简单、常见的具有氧化还原活性的有机分子。它可以得到一个到两个电子，还原生成负离子自由基。同时还可以结合质子，形成酚类物质，如下图所示。

<mark style="background: #FFB8EBA6;">[苯醌还原过程示意图]</mark>

苯醌在其不同的氧化还原态间转换（醌醇循环），从而在生物体内的电荷转移过程中扮演着中间体的角色。而且其氧化还原过程还伴随了质子的转移，因而它同时也作为质子转移的载体。苯醌在生物体内有多种衍生物，最著名的如辅酶 Q（Coenzyme Q），又称作泛醌 Ubiquinone，是呼吸作用电子传递链的中间体，也是重要的抗氧化剂[^1]。其他如质体醌在绿色植物的光合作用中作为电荷传递中间体，而维生素 K 则参与动物体的凝血作用和骨骼代谢等过程。

<mark style="background: #FFB8EBA6;">[苯醌衍生物分子结构示意图：辅酶 Q（泛醌 Ubiquinone）、质体醌（plastoquinone）、维生素 K]</mark>

醌环上的取代基以及共轭基团等差别造成了苯醌衍生物的氧化还原电势的不同以及相应的生物功能的差别。从分子结构与其氧化还原特性间的关系，是一个化学的基本问题。建立微观结构与宏观性质间联系，也是物理化学学习的重点和难点。

本实验结合实验测量和理论计算研究苯醌及其两种衍生物，萘醌和四甲基苯醌（又称杜醌），的氧化还原能力的变化，并考察这种变化的分子结构起源。

物质的氧化还原特性可以用其标准电极电势来衡量。将苯醌的得电子的还原考虑成一个电池的半反应，即苯醌从电极上得到电子被还原。

$$\text{BQ} + \text{e}^- \rightarrow \text{BQ}^{-1}$$

将其与某一参比电极构成一个完整的电池，测量电池的电动势即可得到该反应的相对电极电势。传统上选取标准氢电极（也可以是其他的参比电极），于是苯醌的标准电极电势可以用如下的电池来表示。

$$\text{Pt}|\text{H}_2(\text{g}, 100\text{kPa})|\text{H}^+ + \{a(\text{H}^+) = 1\} \ || \ BQ, BQ^{(-)}|\text{Pt}$$

通常将电极电势写成 $E_{\text{Ox/Red}}$ 的形式，即还原电势（本实验称为氧化还原电势）[^1]。氧化还原电极电势的高低，可作为氧化态物质获得电子被还原成还原态物质这一反应趋向大小的量度。电势越负越难被还原，反过来说也就越容易被氧化。因此只要考察苯醌及其衍生物的相对的氧化还原电势，就可以研究他们的相对氧化还原能力。

另一方面，从热力学角度考虑，也可以研究苯醌及其衍生物还原前后的吉布斯自由能差，根据 $\Delta G = -zFE$ 即可算出电极电势。

本实验将采用电化学测量与理论化学计算相结合方法。实验分为两部分。第一步，采用循环伏安法测量苯醌及其衍生物的标准氧化还原电势。第二步利用量子化学计算软件 Gaussian，结合统计力学原理，计算每个还原反应的吉布斯自由能变，转化为氧化还原电势，从而可以与实验测试结果相比较。再进一步利用理论计算得到的熵、热力学能、分子轨道等信息，分析物质宏观热力学及电化学性质的分子结构基础。

# 实验原理

## 2.1 循环伏安法测量苯醌及其衍生物的氧化还原电势

循环伏安法是常见的电化学研究方法，该方法使用的仪器简单，操作方便、图谱解析直观，在电化学、无机化学、有机化学、生物化学等多研究领域被广泛应用[2, 3]。

<mark style="background: #FFB8EBA6;">[循环伏安法原理示意图]</mark>

循环伏安方法中，工作电极以一定的速率在给定的电压范围内进行连续三角波扫描。电势向阴极方向扫描时，电极中的电子能量升高，注入活性物质，导致其被还原；电势向阳极方向扫描时，电极电子能量降低，电子从电活性物质流向电极，从而被氧化。伴随电势扫描，电活性物种在电极表面交替的被氧化和被还原，在循环伏安图上出现电流方向相反的两个峰，分别为氧化峰和还原峰。

当分子与电极可以进行快速的电子交换，以至于在电压扫描的每个时刻，电极表面的氧化及还原物种的都与电极电势形成平衡，这种情况称为可逆过程。此时氧化还原物种的活度与电极电势符合能斯特方程：

$$E = E^0 + \frac{RT}{nF}\ln\left(\frac{a_{ox}}{a_{red}}\right)$$

式中 $a_{ox}$ 和 $a_{red}$ 分别为氧化和还原物种的活度，$E$ 和 $E^0$ 分别为电极电势和标准电极电势。

对于一个理想的可逆的体系，循环伏安测试中的两个峰的电流大小相等，即 $|i_{p,c}| = |i_{p,a}|$，且对于单电子反应，两峰的峰电位（$E_{p_O}$ 和 $E_{p_R}$）会分裂开 59 mV。氧化峰和还原峰的平均电位称为该循环伏安曲线的半波电位 $E_{1/2}$。此时电极表面的氧化和还原物种浓度相同 $[Ox]=[Red]$，此时的电势称为形式电势。通常可以认为氧化还原物种的活度也近似相同[^2]，即 $a_{ox} \approx a_{red}$，根据能斯特方程，此时的电极电势 $E = E_{1/2} \approx E^0$ [4]。

循环伏安方法测量标准电极电势简便易行，不需严格控制氧化还原物种浓度，数据分析方便。本实验测量苯醌及其衍生物的循环伏安曲线，从曲线中氧化还原峰的峰电流的大小以及峰分裂判断其可逆性，进而就可用其半波电位找到其标准电极电势。

## 2.2 量子化学计算

量子化学计算从量子力学的原理和方法出发，通过解多原子分子体系的 Schrödinger 方程得到分子的本征特性。是化学研究的重要现代化工具 [5]。

依据量子力学理论，已知一个分子体系的基本物理量，包括原子核和电子的质量和电荷，就可以通过求解 Schrödinger 方程得到一个体系的能量和波函数，进而就可以算出分子的所有性质。这一方法不依赖任何其他的测试，只靠基本的原子相关的物理量，通过计算得到分子的性质，因此又称为从头算起（*ab initio*）方法。相比于各种实验方法，理论计算，原则上来说，可以得到更准确、丰富和精细的分子本征特性，是一种强大的理论工具。然而精确求解 Schrödinger 方程，尤其是对于多原子和多电子的体系，在数学方面是异常困难的，必须依赖大量的、复杂的计算。近些年来，随着诸多新的理论和数学方法的出现，各种优秀的量子计算软件不断涌现。同时现代计算机强大的计算能力也使量子化学计算准确程度和效率不断提高，从而可以解决越来越复杂的分子体系。而小型计算机的普及也使得小规模的量化计算变得简便易得，能够处理相对较小的体系。

在理论方面，人们已经开发出了多种方法应对复杂体系求解 Schrödinger 方程的困难。如 Hartree-Fock（HF）方法是一个应对多原子核和多电子体系问题的近似方法，它将多粒子系统的求解问题化为一个求解单粒子波函数的问题。密度泛函理论（DFT）法则用求解电子密度来代替求解波函数，而体系的能量则通过电子密度的泛函（functional）得到。这一方法大大简化了计算，其计算结果相较于 HF 方法精度也更好。

量子化学计算面临的一个主要问题是分子的波函数的数学表达通常没有简单的形式。而分子轨道必须用其他已知的函数的组合来近似描述。这种函数称为基函数，其组合称为基组（basis set）。原则上来说，分子轨道波函数可以由无数多个基函数的组合精确的表示，但这在实际计算中是无法实现的。因此必须找到合适的近似基组，既能足够精确的描述波函数，又不会导致过大的计算量。其中最常用的是用 Gaussian 轨道（GTO）将原子的不同价层用不同数量的 GTO 组合来近似。如 3-21G 表示内核轨道由 3 个 GTO 组成，价层轨道的内层部分由 2 个 GTO 组成，外层由 1 个 GTO 组成。而 6-31G 则表示内核轨道由 6 个 GTO 组成，价层轨道的内层部分由 3 个 GTO 组成，外层由 1 个 GTO 组成。此外原子相互靠近时对电子波函数的极化以及最外围电子的弥散效应也需进行一些校正。如 6-31G+d 就是在 G-31G 的基础上增加了弥散和极化修正，从而可以改善计算的准确度。

在计算出分子轨道和能级后，分子的各种形式的运动的能量，包括平动、转动、振动、电子运动，都可以算出。从而分子的配分函数 $q$ 就可以得到

$$q = \sum_j e^{-\varepsilon_j/(k_B T)}$$

式中 $j$ 表示所有运动形式的所有能量态，$\varepsilon_j$ 为相应的能量，$k_B$ 为玻尔兹曼因子，$T$ 为温度。

依据统计力学原理，有了配分函数，所有的热力学量就都可以算出。例如体系的熵 $S$

$$S = R + R\ln(q) + RT\left(\frac{\partial \ln q}{\partial T}\right)_V$$

式中 $R$ 为理想气体常数，这里考虑的是摩尔熵。

而体系的热力学能 $E$，也即设定基态能量为零时的摩尔内能，也可以算出

$$E = RT^2\left(\frac{\partial \ln q}{\partial T}\right)_V$$

再利用公式 $G = E + RT - TS$，即可算出吉布斯自由能。

本实验采用 Gaussian 软件来进行计算。Gaussian 是一个功能强大的量子化学综合软件包，可以计算很多分子体系的性质，如原子电荷和电势、分子轨道、振动频率、红外和拉曼光谱、核磁性质、热力学性质、反应路径，过渡态等等。我们仅仅使用其一小部分功能。

为了计算苯醌的氧化还原电势，我们首先需要计算苯醌及其还原态半醌负离子自由基的吉布斯自由能差 $\Delta G$。因而计算中将首先计算苯醌的吉布斯自由能（$G_{\text{BQ}}$），再计算半醌负离子自由基的吉布斯自由能 $G_{\text{BQ}^{(-)}}$，后者与前者做差即得到反应的吉布斯自由能差。

$$\Delta G^0 = G^0_{\text{BQ}^{(-)}} - G^0_{\text{BQ}} = -zFE^0$$

这里需要注意的是，这一方法得到的电势是相对于真空能级的，也即是绝对电势。而经典的电极电势都是相对标准氢电极的。他们之间可以用标准氢电极的绝对电势为 4.44 V 进行换算。

# 三、实验

## 3.1 循环伏安法测定苯醌及其衍生物的氧化还原电势

**仪器与试剂**

仪器：电化学工作站，玻璃碳电极，Ag/AgO 参比电极，铂丝对电极，电解池，超声波清洗机，氮气球，打磨工作电极使用的麂皮抛光布和抛光粉。

药品：1,4-苯醌、1,4-萘醌、四甲基-1,4-苯醌（杜醌）的乙腈电解质（四丁基四氟硼酸铵电解质）溶液。

**实验步骤**

1. **准备电极：**
   - A）将对电极和参比电极在乙腈溶剂中超声清洗 3 分钟，擦干备用。
   - B）在麂皮抛光布上加上少许 0.3 微米尺寸的 $\alpha-\text{Al}_2\text{O}_3$ 粉末，滴加少许水使其分散。将玻碳电极垂直于抛光布表面，轻轻施加压力，以画“8”字的方式打磨约 3 分钟。电化学反应都是在电极表面发生的，工作电极的表面是否平整、干净决定了电化学测试的效果，因此工作电极的打磨和清洁是循环伏安法最关键的部分。在水中超声清洗 3 分钟。用蒸馏水冲洗 30 秒以除去 $\text{Al}_2\text{O}_3$ 抛光粉，用氮气枪吹干。放入乙腈超声清洗 30 秒，以除去可能残留的有机杂质，以及上一步可能残留的水。用纸巾轻轻擦干备用。

2. **二茂铁标定（选做）：**
   通过二茂铁标准样检验电极和电解池是否工作正常。

   向电解池中加入已配好的二茂铁-电解质-乙腈溶液。将电极放入电解池，依次连接工作电极，对电极和参比电极。启动电化学工作站，进入程序，设置参数：初电位：0，高电位：0.8，低电位：0.4，扫速：0.1，Segment：8，Sensitivity：$1\times10^{-5}$。开始测量，查看图形分析，观察氧化峰与还原峰的峰电流值以及峰电位。如氧化还原电流大致相等，且氧化还原峰位相差 70 mV 左右，则表明电极过程为理想可逆过程，电极状态良好，电解池工作正常，可进行下一步实验。

3. **测定苯醌及其衍生物的电极电势。**
   用注射器向电解池中加入 ~20 mL 苯醌-乙腈-电解质溶液。由于苯醌及其衍生物都在较低的电势下还原，溶剂中的氧气会对还原造成干扰，因而需要除掉溶剂中的氧气。用氮气球向溶液中通氮 10 分钟除氧。

   打开程序，改变参数，高电位 0，低电位 -1，扫描次数 8 次。点击开始。结束后查看图形，观察氧化还原电流是否大致相等，氧化还原峰位相差是否在 70 mV 左右，判断电极反应是否为理想可逆过程。记录峰电位数据。

   重复步骤 4 和 5 测量萘醌和杜醌电极电势。

   每个实验结束后，开始下一个实验前，均需清洗电极，以保证清洁的电极表面，也防止上一次实验中的电极污染新的溶液。将工作电极在乙腈中超声 30 秒钟，用 $\text{N}_2$ 吹干。对电极和参比电极用乙腈溶剂清洗，擦干备用。

4. **结束工作**
   将电极及电解池用乙腈洗净干燥。关闭电化学工作站。

## 3.2 用 Gaussian 软件计算苯醌及其衍生物的氧化还原电势

1. **构建分子**
   打开 GaussView 5.0 界面。点击工作栏“View”菜单下的 Builder 面板。在面板中鼠标左键点击苯环标志，再在工作窗中点击鼠标左键，创建出苯分子。而后将苯环对位的两个 CH 基团替换成为碳氧双键，点击工具栏上的 clean 按钮得到粗略的分子构型。苯醌的分子结构即构建完成。

2. **优化分子结构和计算分子性质**
   点击菜单栏中的“Calculate”菜单下的 Calculation Setup 选项。在 Job Type 中选 optimization，优化分子构型。在 Method 中，选择计算有机分子最常用的密度泛函理论，即 DFT 方法，B3LYP 杂化泛函。基组我们先选取比较小的 3-21G 基组，以进行快速的结构优化。Solvation 选项选择 CPCM 模型下的乙腈溶剂。点击 submit 按钮提交任务，保存文件后点击 OK 选项开始计算。

   第一次计算完成后，将生成的检查点文件，即 chk 文件，用 GaussView 打开，再次设置计算参数。除了优化结构，我们还需计算分子的振动频率和分子振动的能级等信息，以便计算热力学函数。因而 Job type 选择 optimization+frequency。同时为了增加计算的准确性，Method 设置中选用更大的 6-31G 基组，同时增加扩散和极化效应，即选择“+d”。而后点击 submit 提交，开始第二次计算。

   计算完成后打开 Gaussian 的输出文件，即后缀为 log 的文件。在文件中，热力学数据，包括内能、吉布斯自由能及熵，等已经由软件算出，只需找到并记录即可。点击工作栏的分子轨道编辑选项，可以查看最高占有轨道和最低未占有轨道的能量。

3. **苯醌的还原态的计算**
   打开苯醌的初始输入文件，将 Method 中的电荷处变成 -1，其他参数与苯醌保持相同。同样先在 3-21G 基组下先优化结构，而后在 6-31G+d 下进一步优化结构并计算分子性质。

4. **萘醌和杜醌计算**
   萘醌和杜醌计算过程与苯醌类似，构建分子时萘醌在萘环基础上替换羰基，杜醌在苯醌上将 H 原子替换为甲基即可。
