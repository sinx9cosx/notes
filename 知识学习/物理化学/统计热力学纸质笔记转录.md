# 统计热力学

## 1. 物质体系的微观运动的基本规律

### (分子)基本微观运动形式

平动、转动、振动、电子运动

#### (1) 分子平动

$$\varepsilon_t = \frac{h^2}{8m}\left(\frac{n_x^2}{a^2} + \frac{n_y^2}{b^2} + \frac{n_z^2}{c^2}\right)$$

#### (2) 分子转动

(考虑双原子分子)

(刚性转子模型)
$$\varepsilon_r = \frac{h^2}{8\pi^2I}J(J+1) = \frac{\hbar^2}{2I}J(J+1)$$

$$I = \mu d^2 \quad \mu = \frac{m_1m_2}{m_1+m_2}$$

转动能级的简并度 $g_{(r)} = 2J+1$

简并度：同一个能级对应多个不同微观状态的数量

#### (3) 分子振动

(谐振子模型)
$$\varepsilon_v = (v+\frac{1}{2})h\nu \quad \nu = \text{频率}$$

#### (4) 电子与核运动

只讨论系统中全部粒子的电子与核运动均处于基态。

## 2. 分子不同形式微观运动的独立性与耦合性

(1) 分子的不同形式的微观运动可以独立处理，即微观运动具有独立性。

(2) 能量具有加和性：$\varepsilon_{tot} = \varepsilon_t + \varepsilon_r + \varepsilon_v + \varepsilon_e + \varepsilon_n$

(3) 微观状态：能量在不同运动形式不同粒子上的状况

(4) 耦合性：不同粒子的同一形式的运动的能量可以相互交换；同一粒子的不同形式微观运动的能量也可以相互转换

(5) 独立性是对于一个体系的某一时刻的状态而言
耦合性是对于体系的时间演化而言

## 3. 统计力学解决的问题

(1) 对于一个由大量粒子组成的体系，在相当长的时间尺度上，系统的所有粒子在各个形式的运动能量上的分布状况

(2) 这种分布与宏观系统的性质有什么关系？

## 微观状态

1. **分布**：一组宏观约束条件下，描述粒子在能级上占据数目的集合

2. **微观状态**：系统内所有粒子的状态被完全精确指定的状态

3. **微观状态数 $\omega$**：微观状态的量

### 粒子是否可区分：
(经典统计 $\&$ 量子统计)

1. 经典：局域系统，靠位置坐标区分 (固体)

2. 量子：非局域系统，不能区分 (气体，电子...)
波函数可重叠
晶格上的原子被置换，波函数的重叠积分趋近于0

### $\omega$ 计算：
(不可区分)
一个系统含 $N$ 个粒子，$n$ 个能级，第 $i$ 个能级上粒子数为 $N_i$

$$\omega = \frac{N!}{\prod_i N_i!}$$

除掉 $N!$：能级内部粒子的排列方式不可分辨

## 统计力学基本假设

所有满足系统边界条件的微观状态，其出现概率相等

无论系统初始处于哪个状态，经过足够长时间，一定会遍历所有状态，且在每一个状态上存在的时间相等

具有大的微观状态数的分布状态在体系性质方面起主导作用

---

## 最可几分布

最可几分布：具有最大微观结构数/最大权重的分布方式
可以代表体系，决定了体系的宏观性质

```
低概率状态          自发不可逆           高概率状态
(低微观状态数)      时间演化             (高微观状态数)
      ↑                                    ↑
    低熵态        自发不可逆过程          高熵态
```

① 系统最终必然在最可几分布附近 (到达平衡态)

② 系统自发偏离到较低熵态的时间间隔远比加速回归时间
在有限人类观测尺度上，那极小概率事件等同于不可能事件

---

## 熵

玻尔兹曼方程：$S = k\ln\omega$

- 熵是一个宏观状态的微观结构数的另一种数学描述
- 熵和权重具有物理化学意义上的同质性

### 熵增加原理 (孤立系统)

对于一个宏观系统，如果不受到环境扰动，那么该系统总是自发不可逆地朝熵增加的方向发展。当系统达到最可几宏观状态时，系统的熵达到最大，系统也就达到了平衡点

(等价于) 对孤立系统：$\Delta S \geq 0$

$$
\begin{cases}
\Delta S > 0 & \text{自发进行} \\
\Delta S = 0 & \text{孤立系统处于平衡态}
\end{cases}
$$

(热力学第二定律的原始表述形式)

最可几分布的 $\omega$ 在全部微观状态数中的比例是随体系增大而降低，
为什么可以用最可几分布来描述整个体系？

$\omega$ 几乎全集中在峰值处附近

$S \propto \ln\omega$ \quad $\omega$ 的 $\Rightarrow$ 对 $S$ 贡献小

---

## 温度

熵与物质的质量条件下系统熵与能量的关系

- 物体之间不通过做功的能量传递形式叫做热

- 如果两个系统不处于热平衡：
$$dS_{总} = dS_{热} + dS_{冷} > 0$$

处于平衡：
$$dS_{总} = dS_{热} + dS_{冷} = 0$$
$$dU_{热} + dU_{冷} = 0$$

非热平衡时：
$$dU_1 = -dU_2 > 0 \quad (1\text{吸}\quad 2\text{热})$$
$$dS_1 + dS_2 > 0 \Rightarrow \frac{dS_1}{dU_1} > \frac{dS_2}{dU_2}$$

$$\frac{\partial S_1}{\partial U_1} > \frac{\partial S_2}{\partial U_2}$$

熵对能量偏导：反映的是一个系统熵改变的难易程度
传热过程的方向是熵容易获得能量后熵变的方向

$$\left(\frac{\partial S}{\partial U}\right)_{V,n} \geq 0$$

给定体积和物质的量，系统的熵总是随系统的能量增加而增加

### 广度性质与强度性质

- 广度性质：系统的某个性质正比于系统的大小
- 强度性质：与系统的大小无关

熵和能量是广度性质

$$\left(\frac{\partial U}{\partial S}\right)_{V,n}$$ 是强度性质

### 绝对温度的定义

由上文：$\frac{\partial S_1}{\partial U_1} > \frac{\partial S_2}{\partial U_2} \Leftrightarrow \frac{\partial U_1}{\partial S_1} < \frac{\partial U_2}{\partial S_2}$

定义绝对温度：$T = \left(\frac{\partial U}{\partial S}\right)_{V,n}$

在不改变体积和物质的量的情况下，温度表示系统增加熵
所耗费的能量

- 从低温到高温，系统总熵减小，不会自发发生

### 温度的特性

(1) 温度是系统熵变化难易程度的标度

(2) $T = \left(\frac{\partial U}{\partial S}\right)_{V,n}$

(3) 温度测量系统的冷热程度

(4) 温度决定热传导的方向

---

## 负温度

两个条件：
1. 系统能级存在有限上限
2. 系统处于非平衡激发态，处于高能级的粒子多于处于低能级的粒子数

此时 $U \uparrow$，$\frac{\partial S}{\partial U} < 0$，$\frac{1}{T} < 0$，出现负温度

eg：核自旋系统的粒子数反转

- 能量很低时：粒子大多在低能级，状态数少，熵小
- 能量适中时：粒子在高低能级间均匀分布，$\omega$ 最大
- 能量接近上限时：粒子几乎全在最高能级，$\omega$ 小，$S$ 下降

$$S \uparrow \xrightarrow{T \to \infty} U$$

- $T \to \infty$：顶点附近无论加多少能量，熵几乎不变，意味着系统对能量变化无所谓，$T = \frac{\partial U}{\partial S} \to \infty$
- 热量自发从负温系统流向正温系统

$$\begin{cases}
dS_A + dS_B > 0 \\
dU_A + dU_B = 0
\end{cases}$$

$$T_A = \frac{\partial U_A}{\partial S_A} < 0 \quad T_B = \frac{\partial U_B}{\partial S_B} > 0$$

$$\frac{dU_A}{T_A} + \frac{dU_B}{T_B} > 0 \Rightarrow \frac{dU_A}{T_A} > \frac{dU_A}{T_B} \Rightarrow dU_A < 0$$

能量从 A 流向 B

## 玻尔兹曼分布

(对与恒温热源接触的系统)

- 小系统与大热源、接触达到平衡后，小系统可以在两个能级 ($\varepsilon$ 和 $0$) 上转换 (考虑一个粒子)

大热源能量：$U_{源}$，$U_{源}-\varepsilon$

处于 $\varepsilon$ 和处于 $0$ 的相对概率：

$$P = \frac{P(\varepsilon)}{P(0)} = \frac{W(\varepsilon)}{W(0)} \cdot \frac{W(U-\varepsilon)}{W(U)} = \frac{W(U-\varepsilon)}{W(U)} = \frac{e^{S(U-\varepsilon)/k}}{e^{S(U)/k}}$$

$$= e^{\frac{S(U-\varepsilon)-S(U)}{k}}$$

对 $S(U-\varepsilon)$ 泰勒展开：$S(U-\varepsilon) \approx S(U) - \varepsilon\left(\frac{\partial S}{\partial U}\right)_{V,n} = S(U) - \frac{\varepsilon}{T}$

$$\Rightarrow P = \frac{P(\varepsilon)}{P(0)} = e^{-\frac{\varepsilon}{kT}} \quad \text{(取分子计量)}$$

$$P = \frac{P(\varepsilon)}{P(0)} = e^{-\frac{\varepsilon}{k_BT}} \quad \text{(取摩尔计量)}$$

玻尔兹曼因子

$$E = N_A\varepsilon \quad R = N_Ak \quad \text{阿伏伽德罗公式} \quad k = 1.38 \times 10^{-23} J/K$$

$$\frac{P_{298}(\varepsilon)}{P_{298}(0)} = e^{-\frac{2500}{8.314 \times 298}} = 0.363$$

$$E = \frac{0.363}{0.363+1} \times 100\% = 26.6\%$$

$$Z = 1-26.6\% = 73.9\%$$

$$\frac{P_{77}(\varepsilon)}{P_{77}(0)} = e^{-\frac{250}{8.314 \times 77}}$$

$$E = \frac{0.020}{0.020+1} \times 100\% = 1.9\%$$

$$Z = 1-1.9\% = 98.1\%$$

结论：随着温度升高，系统粒子在高能级分布越多

---

## 玻尔兹曼分布的作用

(1) 体系能级越高，分布越少

(2) 温度越高，系统在高能级分布越多，系统的内能越多

(3) 能级间的能量差越大，系统在高能级分布的越少

---

## 玻尔兹曼分布的特性

(1) 系统到达平衡时，系统分子在不同能级上满足玻尔兹曼分布
(在给定温度条件下，玻尔兹曼分布是系统的最可几分布)

(2) 玻尔兹曼是严格定域的指数分布

(3) 对于孤立系统，玻尔兹曼分布对应于熵最大宏观状态

(4) 玻尔兹曼分布的任何两个能级之间，都满足玻尔兹曼因子，能量项为两个能级的差值

---

## $kT$ ($RT$) 的物理意义

从玻尔兹曼分布的角度，若两个能态的能量差 $= kT$

$$P = \frac{P(\varepsilon)}{P(0)} = e^{-\frac{kT}{kT}} = e^{-1} = \frac{1}{2.72}$$

则两个态之间的分布的粒子数之比为 $1:2.72$

若能量高于 $kT$，则其分布占比相比于 $2.72$ 更低，
低于 $kT$ 则会升高

因此可以把 $kT$ 理解为一个能量标杆。体系高于这个能量的态，很少有分布；低于这个态则容易有更多的分布

$kT$ 带有能量量纲，因指数项必须无量纲

## 最大值法推导玻尔兹曼分布

target：在粒子数固定，体系能量固定的条件下找到微观状态数目的最大值

$$\omega = N!\prod_i \frac{g_i^{n_i}}{n_i!} \quad g_i: \text{能级} \varepsilon_i \text{的简并度}$$

约束条件：
$$N = \sum_i n_i \quad \text{粒子数}$$
$$U = \sum_i n_i\varepsilon_i \quad \text{总能量}$$

(解释)：处理对象是经典粒子，一个微观态可以有无数个粒子
$g_i^{n_i}$：$n_i$ 个粒子同时独立选择 $g_i$ 个态
高温低密度下：$g_i$ 极大，$n_i$ 较小

Stirling 近似：$\ln N! \approx N\ln N - N$

转化为求 $\ln\omega$ 的极值：$\ln\omega = N\ln N - N + \sum_i (n_i\ln g_i - n_i\ln n_i + n_i)$

$$= N\ln N + \sum_i (n_i\ln g_i - n_i\ln n_i)$$

设 $Z = \ln\omega + \alpha(\sum_i n_i - N) + \beta(\sum_i n_i\varepsilon_i - U)$

$$= N\ln N + \sum_i (n_i\ln g_i - n_i\ln n_i) + \alpha(\sum_i n_i - N) + \beta(\sum_i n_i\varepsilon_i - U)$$

令 $\frac{\partial Z}{\partial n_i} = 0$ (对一个特定能级求偏导，无 $\sum$)

$$\ln g_i - \ln n_i - 1 + \alpha + \beta\varepsilon_i = 0 \quad \ln n_i = \ln g_i + 1 + \alpha + \beta\varepsilon_i$$

$$n_i = e^{1+\alpha}g_i e^{\beta\varepsilon_i} = e^{\alpha'}g_i e^{\beta\varepsilon_i} \quad (i=1,2,3\cdots)$$

$$e^{\alpha'} = \frac{N}{\sum_i g_i e^{\beta\varepsilon_i}} \quad n_i = N\frac{g_i e^{\beta\varepsilon_i}}{\sum_i g_i e^{\beta\varepsilon_i}}$$

(推导 $\beta$)

$$\ln\omega = N\ln N + \sum_i (n_i\ln g_i - n_i\ln n_i)$$

$n_i = e^{\alpha}g_i e^{\beta\varepsilon_i}$ 代入

$$\ln\omega = N\ln N + \sum_i [e^{\alpha}g_i e^{\beta\varepsilon_i}\ln g_i - e^{\alpha}g_i e^{\beta\varepsilon_i}(\alpha + \ln g_i + \beta\varepsilon_i)]$$

$$= N\ln N + \sum_i [-\alpha n_i - \beta n_i\varepsilon_i]$$

$$= N\ln N - \alpha N - \beta U \quad \text{令} \alpha' = \alpha - \ln N \quad \ln\omega = \alpha'N - \beta U$$

$$S = k\ln\omega = k(\alpha'N - \beta U)$$

$$\left(\frac{\partial S}{\partial U}\right)_N = -k\beta = \frac{1}{T} \Rightarrow \beta = -\frac{1}{kT}$$

$$\Rightarrow n_i = N\frac{g_i e^{-\varepsilon_i/kT}}{\sum_i g_i e^{-\varepsilon_i/kT}} \quad \text{第 $i$ 个量子态中分布的粒子的概率}$$

使 $\omega$ 取极大值，该分布为最概然分布，称为玻尔兹曼分布

## 配分函数

$$q = \sum_i g_i e^{-\varepsilon_i/kT}$$

### ① 系统总能量与配分函数的关系

$$U = \sum_i n_i\varepsilon_i = \sum_i N\frac{g_i e^{-\varepsilon_i/kT}}{q}\varepsilon_i \quad (g_i \to ?)$$

$$= \frac{N}{q}\sum_i g_i\varepsilon_i e^{-\varepsilon_i/kT}$$

$$= \frac{N}{q}kT^2\sum_i g_i\frac{d(e^{-\varepsilon_i/kT})}{dT}$$

$$d(e^{-\varepsilon_i/kT}) = e^{-\varepsilon_i/kT} \cdot \left(-\frac{\varepsilon_i}{k}\right) \cdot \left(-\frac{k}{T^2}\right)dT = e^{-\varepsilon_i/kT}\frac{\varepsilon_i}{kT^2}dT$$

$$U = Nk\frac{q}{q}kT^2\frac{dq}{dT} = NkT^2\frac{d\ln q}{dT} \quad \text{相对于基态能}$$

总能量 $U = NkT^2\frac{d\ln q}{dT} + U(0)$

$$= NkT^2\frac{d\ln q^0}{dT} + U(0)$$

热能 | 基态能

系统的热能是由系统微观运动引起的一定比 $U(0)$ 高的分子分布到激发态所致，因此系统由微观运动形式对应的能级特点及在能级上的分布情况决定了系统的热能和对应温度

基态能的这部分对热能无贡献

$$\uparrow \uparrow \quad U \uparrow$$

eg：$A$ 体系 $\longrightarrow B$

$$U_A^m(T_A) + U_B^m = U_B^m(T_B) + U_B^0$$

## 配分函数的析因子性质

粒子运动：独立的平动、转动、振动、电子运动和核子运动

$$\varepsilon_i = \varepsilon_{t,i} + \varepsilon_{r,i} + \varepsilon_{v,i} + \varepsilon_{e,i} + \varepsilon_{n,i}$$

$$q = \sum_i g_i e^{-\varepsilon_i/kT} = \sum_i g_i e^{-(\varepsilon_{t,i}+\varepsilon_{r,i}+\varepsilon_{v,i}+\varepsilon_{e,i}+\varepsilon_{n,i})/kT}$$

$$= \sum_i (e^{-\varepsilon_{t,i}/kT} \cdot e^{-\varepsilon_{r,i}/kT} \cdot e^{-\varepsilon_{v,i}/kT} \cdot e^{-\varepsilon_{e,i}/kT} \cdot e^{-\varepsilon_{n,i}/kT})$$

(?) 补证：$\sum_i e^{\varepsilon_i}e^{\varepsilon_i'} \stackrel{?}{=} (\sum_i e^{\varepsilon_i})(\sum_i e^{\varepsilon_i'})$

全部展开(?)
左 $= e^{\varepsilon_1}e^{\varepsilon_1'}+e^{\varepsilon_1}e^{\varepsilon_2'}+\cdots$
$+e^{\varepsilon_2}e^{\varepsilon_1'}+\cdots$

右 $= (e^{\varepsilon_1}+\cdots+e^{\varepsilon_n})(e^{\varepsilon_1'}+\cdots+e^{\varepsilon_n'})$
$=$ 左

即得 $q = q_t \cdot q_r \cdot q_v \cdot q_e \cdot q_n$

## 配分函数的意义

有效可及状态数
连接宏观与微观的热力学母函数
归一化因子：它把总概率"1"按玻尔兹曼权重分配给每一个微观状态

若 $\varepsilon_i = 0$ 所有能态能量相同，则
$q = \sum_i 1$，但 $e^{-\varepsilon_i/k_BT}$ 对高能态打折扣

## 能量的零点的选择对配分函数的影响

设某运动基态能设为 $\varepsilon_0$ 作为零点，则 $\varepsilon_i^0 = \varepsilon_i - \varepsilon_0$

$$q = \sum_i g_i e^{-\varepsilon_i/kT} = \sum_i g_i e^{-(\varepsilon_i^0+\varepsilon_0)/kT}$$

$$= e^{-\varepsilon_0/kT} \cdot \sum_i g_i e^{-\varepsilon_i^0/kT} = e^{-\varepsilon_0/kT} \cdot q^0$$

$$q = e^{-\varepsilon_0/kT} \cdot q^0 \quad q^0 = \sum_i g_i e^{-\varepsilon_i^0/kT}$$

平动：$\varepsilon_{t,0} = 0$ (三维势箱) $\Rightarrow q_t^0 = q_t$

转动：$\varepsilon_{r,0} = 0$ (理想刚体) $\Rightarrow q_r^0 = q_r$ \quad $q_r = \frac{8\pi^2IkT}{\sigma h^2} = \frac{T}{\sigma\Theta_r}$

振动：$\varepsilon_{v,0} = \frac{h\nu}{2}$ (谐振子) $\Rightarrow q_v^0 = e^{h\nu/2kT} \cdot q_v = e^{\Theta_v/2T} \cdot q_v$

$\Theta_v = h\nu/k$

通常温度下，$q_v^0$ 与 $q_v$ 的区别不能忽略

能量的零点的选择会影响配分函数的数值，但不影响玻尔兹曼分布的能级 (状态) 分布数

$$n_i = \frac{N}{q}g_i e^{-\varepsilon_i/kT} = \frac{N}{q^0 e^{-\varepsilon_0/kT}} \cdot g_i e^{-(\varepsilon_i^0+\varepsilon_0)/kT} = \frac{N}{q^0}g_i e^{-\varepsilon_i^0/kT}$$

## 平动配分函数

$$\varepsilon_{n_x,n_y,n_z} = \varepsilon_{n_x} + \varepsilon_{n_y} + \varepsilon_{n_z}$$

$$q_t = q_{t,x} \cdot q_{t,y} \cdot q_{t,z}$$

$$q_{t,x} = \sum_{n=1}^{\infty} e^{-\frac{h^2}{8ma^2kT}n^2}$$

由于平动能级间距很小，加和可用积分近似

$$q_{t,x} = \int_0^{\infty} e^{-\frac{h^2}{8ma^2kT}n^2} dn = \sqrt{\frac{2\pi mkT}{h^2}}a$$

$$\int_0^{\infty} e^{-\alpha x^2}dx = \frac{1}{2}\sqrt{\frac{\pi}{\alpha}}$$

$q_{t,y}$、$q_{t,z}$ 同理

$$q_t = \left(\frac{2\pi mkT}{h^2}\right)^{3/2}(abc) = \left(\frac{2\pi mkT}{h^2}\right)^{3/2}V$$

$V = abc$ 系统体积

$$q_t = f_t^3 \quad f_t = \left(\frac{2\pi mkT}{h^2}\right)^{3/2}V^{1/3}$$

(相对原子质量 $\times$ 摩尔质量)

$$m = \frac{M}{N_A}$$

## 转动配分函数

考虑双原子分子，用刚性转子模型

$$\varepsilon_r = J(J+1)\frac{h^2}{8\pi^2I} = J(J+1)\frac{\hbar^2}{2I}$$

$$g_r = 2J+1$$

$$q_r = \sum_{J=0}^{\infty} (2J+1)e^{-J(J+1)\frac{h^2}{8\pi^2IkT}}$$

$$\Theta_r = \frac{h^2}{8\pi^2Ik}$$ 转动特征温度 (通常温度是否足以"转动充分)

积分近似(高温)：
$$q_r = \int_0^{\infty} (2J+1)e^{-J(J+1)\frac{\Theta_r}{T}}dJ$$

令 $x = J(J+1)$

$$q_r = \int_0^{\infty} e^{-\frac{\Theta_r}{T}x}dx = \frac{T}{\Theta_r} = \frac{8\pi^2IkT}{\sigma h^2}$$

$$J=0 \to J=1$$
$$\Delta\varepsilon_{0\to 1} = \frac{h^2}{2I} \times 2 - 0 = \frac{h^2}{I} = 2k\Theta_r \quad \text{对比环境热能} kT$$

当 $T \ll \Theta_r$：环境热能不足以提供 $2k\Theta_r$，分子几乎全部在 $J=0$
转动自由度被冻结

当 $T \gg \Theta_r$：大量转动能级被激发
(能级间隔) $\Delta\varepsilon \ll kT$ 能级几乎连续
在 $kT$ 以上

## 分子对称数

异核双原子分子：$\sigma = 1$ 旋转 $360°$ 重合一次

同核双原子分子：$\sigma = 2$ 旋转 $180°$ 重合一次

这里粒子不可区分，除以 $\sigma$ 来修正经典结果

$$q_r = \frac{T}{\Theta_r\sigma} = \frac{8\pi^2IkT}{h^2\sigma}$$

每个转动自由度的配分函数为 $f_r = q_r^{1/2} = \left(\frac{T}{\Theta_r\sigma}\right)^{1/2}$

第 $J$ 个能级的粒子的比例：

$$f_J = \frac{(2J+1)e^{-J(J+1)\frac{\Theta_r}{T}}}{q_r} = (2J+1)\frac{\Theta_r}{T}e^{-J(J+1)\frac{\Theta_r}{T}}$$

### 自由度

线性分子 $\sigma = 2$，只有两个相互垂直方向的转动有效

非线性分子 $\sigma = 3$，绕 $x,y,z$ 三个轴转动惯量都较大，有3个自由度

## 振动配分函数

$$\varepsilon_{v,i} = (v+\frac{1}{2})h\nu \quad \text{(非简并)}$$

$$q_v = \sum_{v=0}^{\infty} g_{v,i}e^{-\varepsilon_{v,i}/kT} = \sum_{v=0}^{\infty} e^{-(v+\frac{1}{2})h\nu/kT}$$

$$= e^{-h\nu/2kT}\sum_{v=0}^{\infty} e^{-vh\nu/kT}$$

令 $\Theta_v = h\nu/k$ 振动特征温度

$$q_v = e^{-\Theta_v/2T}\sum_{v=0}^{\infty} e^{-v\Theta_v/T}$$

($e^{-\Theta_v/T}$ 的等比级数)

$0 < e^{-\Theta_v/T} < 1$ 级数收敛

$$q_v = \frac{e^{-\Theta_v/2T}}{1-e^{-\Theta_v/T}} = \frac{1}{e^{\Theta_v/2T}-e^{-\Theta_v/2T}}$$

一维谐振子能级非简并，每个振动自由度的配分函数 $f_v = q_v$

以基态能级的能量为 0 时：

$$q_v^0 = e^{\varepsilon_{v,0}/kT} \times q_v = e^{\Theta_v/2T} \times \frac{1}{e^{\Theta_v/2T}-e^{-\Theta_v/2T}} = \frac{1}{1-e^{-\Theta_v/T}}$$

eg：$CO_2$：$3\times 3-5 = 3+2 = 4$
eg：$H_2O$：$3\times 3-3 = 3+3 = 3$

振动自由度计算：一个含 $N$ 个原子的分子
总自由度 $3N$
振动自由度 $= 3N - \text{平动} - \text{转动}$

## 特征温度的物理意义

- $\frac{e^{-\varepsilon_i/k_BT}}{q} = P_i$ 即第 $i$ 个能级上的粒子在全部粒子中所占的比例。因而 $e^{-\varepsilon_i/k_BT}$ 表示没有被 $q$ 归一化的能级 $i$ 上的粒子存在的概率。

- $\frac{\varepsilon_i}{k_BT}$ 因而是没有量纲的，$\frac{\varepsilon_i}{k_B}$ 与温度具有相同的量纲，即 $\frac{\varepsilon_i}{k_B}$ 是某个温度。

- 当 $\frac{\varepsilon_i}{k_B}$ 与 $T$ 相等时，$e^{-\varepsilon_i/k_BT} = \frac{1}{e}$。因此 $\frac{\varepsilon_i}{k_B}$ 的意义在于，当体系的温度达到此温度时，处于这个能量上的粒子的概率 (未归一化的) 达到了 $\frac{1}{e}$

- 特征温度是能量的另一种表达形式，特征温度越高，能量越高。在转动和振动中，特征温度是两种运动形式的成分能量的单位。特征温度代表了能量间隔的大小。转动特征温度低，表明其能级差小，分子更容易分布到高能级，因而其配分函数，熵和热能都更大。

$$q_n = g_{n,0}e^{-\varepsilon_{n,0}/kT}$$

$$q_n^0 = g_{n,0} = \text{常数}$$

## 电子运动的配分函数

只考虑粒子的电子运动全部处于基态的情况

$$q_e = g_{e,0}e^{-\varepsilon_{e,0}/kT}$$

$$q_e^0 = g_{e,0}e^{\varepsilon_{e,0}/kT} = g_{e,0} = \text{常数}$$

分子基态稳定离子的基态能级几乎总是简并的，$g_{e,0} = 1$

例如：$O_2$：2个未成对电子 $\to g_{e,0} = 3$
$NO$：存在一个单电子 $\to g_{e,0} = 2$

(GS 解释中)：$g_{e,0} = 2S+1$ \quad $S$ 为总自旋量子数

正常情况下，所有电子两两配对，自旋相反，$S = 0$：
$g_{e,0} = 2(0)+1 = 1$

$O_2$ 三重态分子：$S = \frac{1}{2} \times 2 = 1$
$g_{e,0} = 2(1)+1 = 3$

$NO$：$S = \frac{1}{2}$
$g = 2 \times (\frac{1}{2}) + 1 = 2$

## 能量均分原理

能量均分原理：(统计力学的一个推论)：在温度足够高 (能级近似连续) 时，热力学的每个平方项对能量的贡献为 $\frac{1}{2}kT$

## 不同微观运动形式对热能的贡献

$$U = U_t + U_r + U_v + U_e + U_n$$

$$= NkT^2\left(\frac{\partial\ln q_t}{\partial T}\right) + NkT^2\left(\frac{\partial\ln q_r}{\partial T}\right) + NkT^2\left(\frac{\partial\ln q_v}{\partial T}\right)$$

$$+ NkT^2\frac{\partial\ln q_e}{\partial T} + NkT^2\frac{\partial\ln q_n}{\partial T}$$

以各运动形式的基态为能量零点：

$$U = NkT^2\left(\frac{\partial\ln q^0}{\partial T}\right)_V + \frac{N\varepsilon_0}{U_0 \text{系统全处于} 0 \text{时的热力学能}}$$

$$U_t^0 \approx U_t \quad U_r^0 = U_r \quad U_v^0 = U_v - \frac{Nh\nu}{2}$$

$$U_e^0 = 0 \quad U_n^0 = 0$$

### (1) $U_t^0$ 平动热能

$$U_t^0 = NkT^2\left(\frac{\partial\ln q_t^0}{\partial T}\right)_V$$

$$= NkT^2\left[\frac{\partial}{\partial T}\ln\left(\frac{2\pi mkT}{h^2}\right)^{3/2}V\right]_V$$

$$= \frac{3}{2}NkT$$

或 $U_{t,m}^0 = \frac{3}{2}RT$

- 每个平方自由度的平均能量 $\frac{1}{2}kT$，符合能量均分原理
- 理想气体内能只与温度有关
(每种运动形式的配分函数都与温度有关)

### (2) $U_r^0$ 转动热能

$$U_r^0 = U_r = NkT^2\frac{\partial\ln q_r}{\partial T} = NkT^2\frac{\partial}{\partial T}\ln\left(\frac{T}{\Theta_r\sigma}\right) = NkT$$

或 $U_{r,m}^0 = RT$

非线性分子：$U_{r,m}^0 = \frac{3}{2}RT$

$$\varepsilon_r = \frac{1}{2}I\omega_1^2 + \frac{1}{2}I\omega_2^2 \quad \text{贡献} 2 \times \frac{1}{2}kT = kT$$

非线性分子：转动可分解为绕三个主轴转动，有3个平方项

为什么符合能量均分原理

### (3) $U_v^0$ 振动热能

$$U_v^0 = NkT^2\frac{\partial\ln q_v^0}{\partial T} = NkT^2\frac{\partial}{\partial T}\ln\left(\frac{1}{1-e^{-\Theta_v/T}}\right)$$

$$= NkT^2 \cdot \frac{-\Theta_v/T^2 \cdot e^{-\Theta_v/T}}{1-e^{-\Theta_v/T}} = Nk\Theta_v\frac{1}{e^{\Theta_v/T}-1} = \frac{Nk\Theta_v}{e^{\Theta_v/T}-1}$$

① $\Theta_v \gg T \Rightarrow U_v^0 \approx 0$ 振动能级不开放，分子处于振动基态

② $\Theta_v \ll T$ 或 $\Theta_v/T \ll 1 \Rightarrow U_v^0 \approx NkT$

振动能级完全开放，符合能量均分原理

泰勒展开：$e^{\Theta_v/T}-1 \approx \frac{\Theta_v}{T}$

$$U_v^0 = Nk\Theta_v\left(\frac{T}{\Theta_v}\right) = NkT$$

## 不同微观运动形式对摩尔定容热容的贡献

$$U_m = U_m^0 + N\varepsilon_0$$

$$C_{V,m} = \left(\frac{\partial U_m}{\partial T}\right)_V = \left(\frac{\partial U_m^0}{\partial T}\right)_V$$

$\Rightarrow$ 系统的摩尔定容热容与零点能的选取无关

$$C_{V,m} = C_{V,t} + C_{V,r} + C_{V,v} + C_{V,e} + C_{V,n}$$

(1) 若粒子的核及电子运动均不开放，则其对摩尔定容热容无贡献
$$C_{V,e} = 0 = C_{V,n}$$

(2) 平动和转动的贡献：
$$C_{V,t} = \left(\frac{\partial U_{t,m}^0}{\partial T}\right)_V = \frac{3}{2}R$$

$$C_{V,r} = \left(\frac{\partial U_{r,m}^0}{\partial T}\right)_V = R$$

(3) 振动的贡献：
$$C_{V,v} = \left(\frac{\partial U_{v,m}^0}{\partial T}\right)_V = R\left(\frac{\Theta_v}{T}\right)^2 \cdot e^{\Theta_v/T} \frac{1}{(e^{\Theta_v/T}-1)^2}$$

① $\Theta_v \gg T \Rightarrow C_{V,v} \approx 0$

② $\Theta_v \ll T \Rightarrow C_{V,v} \approx R$

## 推论

(粒子及核与电子运动均处于基态，振动能级不开放)

单原子气体 $C_{V,m} = \frac{3}{2}R$

双原子气体 $C_{V,m} = \frac{5}{2}R$

一般地，对双原子气体，$C_{V,m}$ 是 $T$ 的函数
$$\frac{5}{2}R \leq C_{V,m}(T) \leq \frac{7}{2}R$$

## 配分函数与熵的关系

### 非简并态

$$S = k\ln\omega = k\ln\frac{N!}{\prod_i n_i!}$$

$$= k(N\ln N - \sum_i n_i\ln n_i)$$

$$= k\sum_i (n_i\ln N - n_i\ln n_i) = k\sum_i n_i\ln\frac{N}{n_i}$$

$$= -k\sum_i n_i\ln\frac{n_i}{N} = -k\sum_i n_i\ln\frac{g_i e^{-\varepsilon_i/kT}}{q}$$

$$= -k\sum_i n_i(\ln\frac{g_i}{q} - \frac{\varepsilon_i}{kT})$$

$$= -k(\frac{q}{q}\sum_i n_i\varepsilon_i + \ln\frac{q}{q}\sum_i n_i)$$

$$= \frac{U}{T} + kN\ln q \quad \text{定域子系统的熵}$$

### 简并态

$$n_i = N\frac{g_i e^{-\varepsilon_i/kT}}{\sum_i g_i e^{-\varepsilon_i/kT}}$$

$$S = k(N\ln N + \sum_i n_i\ln g_i - \sum_i n_i\ln n_i)$$

$$= -k\sum_i n_i(\ln N + \ln g_i - \ln n_i)$$

$$= -k\sum_i n_i\ln\frac{N}{n_i g_i}$$

$$= -k\sum_i n_i\ln\frac{\sum_i g_i e^{-\varepsilon_i/kT}}{N g_i e^{-\varepsilon_i/kT}}$$

$$= -k\sum_i n_i\ln\frac{q}{N g_i e^{-\varepsilon_i/kT}}$$

$$= -k\sum_i n_i(\ln\frac{q}{N} + \ln g_i - \frac{\varepsilon_i}{kT})$$

$$= \frac{U}{T} + kN\ln\frac{q}{N} + kN$$

---

## 离域子系统的熵

全同粒子不可区分，修正：$\omega' = \omega/N!$

$$S = k\ln\omega' = \frac{U}{T} + Nk\ln q - k\ln N!$$

$$= \frac{U}{T} + Nk\ln\frac{q}{N} + Nk$$

系统的熵与能量零点选择无关

定域子系统：$S = Nk\ln q + \frac{U}{T}$

$$= Nk\ln(e^{-\varepsilon_0/kT} \cdot q^0) + \frac{U}{T}$$

$$= -\frac{N\varepsilon_0}{T} + Nk\ln q^0 + \frac{U}{T}$$

$$= \frac{U_0}{T} + Nk\ln q^0$$

离域子系统 $S = \frac{U_0}{T} + Nk\ln q^0 + Nk$

## 系统熵的分解

$$S = S_t + S_r + S_v + S_e + S_n$$

(差别)
定域子系统的熵与离域子系统的熵体现在平动熵：

$$S_t(\text{离}) = \frac{U_t^0}{T} + Nk\ln\frac{q_t^0}{N} + Nk$$

$$S_t(\text{定}) = \frac{U_t^0}{T} + Nk\ln q_t^0$$

统计熵：平动熵 + 转动熵 + 振动熵

光谱熵：(其参数可通过光谱得到) 转动熵、振动熵
微波谱 红外/拉曼

$$S_t = Nk\ln\frac{q_t^0}{N} + \frac{U_t^0}{T} + Nk$$

$$= Nk\ln\frac{(2\pi mkT)^{3/2}V}{Nh^3} + \frac{3}{2}\frac{NkT}{T} + Nk$$

$$= Nk\ln\frac{(2\pi mkT)^{3/2}V}{Nh^3} + \frac{5}{2}Nk$$

$$S_r = Nk\ln q_r^0 + \frac{U_r^0}{T} = Nk\ln\frac{T}{\Theta_r\sigma} + Nk$$

$$S_v = Nk\ln q_v^0 + \frac{U_v^0}{T} = Nk\ln\frac{1}{1-e^{-\Theta_v/T}} + \frac{Nk\Theta_v}{T(e^{\Theta_v/T}-1)}$$

只有 $S_t$ 修正 ($N!$)：平动配分函数描述了分子位置的交换产生新的状态
转动、振动只描述单个分子内部状态，交换内部状态伴随位置交换已被 $N!$ 修正

## Gibbs 熵

$$N = \sum_i n_i \quad p_i = \frac{n_i}{N} \quad \sum_i p_i = 1 \quad \omega = \frac{N!}{\prod_i n_i!}$$

$n_i$：(能量态) 每个量子态上的粒子数

$$S = k\ln\omega \quad \text{(Boltzmann 熵)} = k\ln(N\ln N - \sum_i n_i\ln n_i)$$

$$= k\sum_i n_i(\ln N - \ln n_i) = -kN\sum_i \frac{n_i}{N}\ln\frac{n_i}{N}$$

$$= -kN\sum_i p_i\ln p_i$$

在恒温恒容下，这个 $p_i$ 是玻尔兹曼分布 $p_i = \frac{e^{-\varepsilon_i/kT}}{q}$

Boltzmann 熵 $S = k\ln\omega$ (由 Gibbs 导出)

适用条件：微正则系综，(孤立系统，$N,U,V$ 固定)
核心假设：等概率原理

### Gibbs 熵

适用条件：任意，甚至非平衡态
核心假设：无 $p_i$ 可以是符合任意条件的概率分布

$$S = -kN\sum_i p_i\ln p_i \quad \text{(定义式)}$$

Gibbs 熵 max = Boltzmann 熵

系统完全确定的 $p_i = 1$，$S_G = 0$

## 香农信息熵

$$H = -\sum_i p_i\log_2 p_i$$

- $p_i$ 是信息源发出某个信号 $i$ 的概率
- 衡量的是平均每个信号携带的信息量
- 信息量 $I_i = -\log_2 p_i$ (bit)
- 联系：Gibbs 熵是信息熵在物理系统中的特例
  数学结构一致，但约束条件不同
  统计力学中，$p_i$ 由物理定律决定
  信息论中，$p_i$ 是信息的概率分布
- 对系统微观状态的无知 (Shannon 熵) 等价于系统可能处于更多微观状态的热力学熵

## Gibbs 悖论思考

1. 你的主观吉布斯熵是不是"错误的"？
你设想的场景是：系统客观上已处于一个确定的微观态，但你不知道，所以你假设了一个概率分布 (比如等概率)，算出了一个大的吉布斯熵。

答案：这个熵不是"错误"，而是"在给定信息约束下的最优推断"。

物理学不是上帝视角，我们永远无法直接感知系统的微观状态。我们能测量的只能是少数几个宏观变量 (温度、压力、体积等)。任何熵的计算，都是基于我们实际拥有的信息做出的。

- 客观真实状态：系统是某个具体的 $i^*$，它的"真实熵"是 0。
- 你的计算：你测了温度 $T$，知道系统平均能量是 $U$。基于这个约束，最无偏的概率分布是玻尔兹曼分布，算出的熵是 $S(T)$。
- 这个 $S(T)$ 的意义：它衡量的是"在所有与测量值 $U$ 兼容的微观状态中，这个系统的不确定性"。

2. 当你知道确切状态时，熵为 0，这矛盾吗？
"当我完全知道了这个系统的状态，香农熵和热力学熵都变为 0，也就是客观上系统的熵吗？"

完全正确。这揭示了熵的一个根本属性：熵不是系统固有的绝对属性，而是系统状态相对于描述者信息的属性。

所以，主观推测的吉布斯熵不是计算错误，而是量化了你缺失的信息。
