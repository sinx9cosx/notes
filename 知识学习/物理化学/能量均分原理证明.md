# 1. 问题设定

能量均分原理是**经典统计力学**（能级近似连续、高温极限）的推论。用**正则系综**（canonical ensemble，系统与温度为 $T$ 的大热源接触达平衡）推导。

系统的哈密顿量 $H(\mathbf{p},\mathbf{q})$ 中，若某个自由度以**平方项**形式出现（例如 $\frac{p_i^2}{2m}$ 或 $\frac{1}{2}kq_i^2$），则该自由度在热平衡时的平均能量为 $\frac{1}{2}kT$。

# 2. 正则系综的概率分布

系统处于相空间体积元 $d\mathbf{p}\,d\mathbf{q}$ 中的概率正比于玻尔兹曼因子：

$$P(\mathbf{p},\mathbf{q})\,d\mathbf{p}d\mathbf{q} \propto e^{-\beta H(\mathbf{p},\mathbf{q})}\,d\mathbf{p}d\mathbf{q}, \qquad \beta \equiv \frac{1}{kT}$$

> [!notes]
> $H(p,q)=T(p)+V(q)$是用广义坐标q和广义动量p表示的系统总能量。
> 玻尔兹曼因子描述的是系统在像空间中某个点（p,q）出现的相对概率。
> 在哈密顿力学框架下（p,q）是一对共轭变量，dpdq相空间体积元在时间演化下保持不变（刘维尔定理），这是等概率原理成立的数学基础。

归一化后：
$$P(\mathbf{p},\mathbf{q}) = \frac{e^{-\beta H}}{Z}, \qquad Z = \int e^{-\beta H}\,d\mathbf{p}d\mathbf{q}$$
其中 $Z$ 是经典配分函数（相空间积分）。

# 3. 平方项的平均值

设哈密顿量中某个平动自由度对应的动能为 $H_1 = \frac{p_x^2}{2m}$（一维平动动能），其余部分记为 $H'$，且 $H_1$ 与 $H'$ 独立（可分离）：
$$H = \frac{p_x^2}{2m} + H'$$
该动能项的平均值为：[[量子力学#一维势阱]]
$$\langle \frac{p_x^2}{2m} \rangle = \frac{\displaystyle\int \frac{p_x^2}{2m}\,e^{-\beta\left(\frac{p_x^2}{2m} + H'\right)}\,dp_x\,d\mathbf{p}'d\mathbf{q}}{\displaystyle\int e^{-\beta\left(\frac{p_x^2}{2m} + H'\right)}\,dp_x\,d\mathbf{p}'d\mathbf{q}}$$

由于 $e^{-\beta(\frac{p_x^2}{2m} + H')} = e^{-\beta p_x^2/2m} \cdot e^{-\beta H'}$，分子和分母中 $H'$ 部分的积分相互抵消：
$$\langle \frac{p_x^2}{2m} \rangle = \frac{\displaystyle\int_{-\infty}^{\infty} \frac{p_x^2}{2m}\,e^{-\beta p_x^2/2m}\,dp_x}{\displaystyle\int_{-\infty}^{\infty} e^{-\beta p_x^2/2m}\,dp_x}$$

# 4. 高斯积分求值

令 $\alpha = \frac{\beta}{2m} = \frac{1}{2mkT}$
**分母（归一化积分）：**
$$I_0 = \int_{-\infty}^{\infty} e^{-\alpha p_x^2}\,dp_x = \sqrt{\frac{\pi}{\alpha}}$$
**分子：**

$$I_2 = \int_{-\infty}^{\infty} p_x^2\,e^{-\alpha p_x^2}\,dp_x$$
利用 $p_x^2 e^{-\alpha p_x^2} = -\frac{\partial}{\partial\alpha}e^{-\alpha p_x^2}$：
$$I_2 = -\frac{\partial}{\partial\alpha}\int_{-\infty}^{\infty} e^{-\alpha p_x^2}\,dp_x = -\frac{\partial}{\partial\alpha}\sqrt{\frac{\pi}{\alpha}} = \frac{1}{2}\sqrt{\frac{\pi}{\alpha^3}}$$
因此：
$$\langle \frac{p_x^2}{2m} \rangle = \frac{1}{2m} \cdot \frac{I_2}{I_0} = \frac{1}{2m} \cdot \frac{\frac{1}{2}\sqrt{\pi/\alpha^3}}{\sqrt{\pi/\alpha}} = \frac{1}{2m} \cdot \frac{1}{2\alpha} = \frac{1}{2m} \cdot \frac{1}{2\cdot(1/2mkT)} = \frac{1}{2}kT$$

# 5. 推广到任意平方项

对于任意一个以平方项形式出现在哈密顿量中的自由度 $A\xi^2$（$\xi$ 可以是动量 $p$ 或坐标 $q$，$A$ 为常数），上述推导完全同理：

$$\langle A\xi^2 \rangle = \frac{\displaystyle\int A\xi^2 e^{-\beta A\xi^2}\,d\xi}{\displaystyle\int e^{-\beta A\xi^2}\,d\xi} = \frac{1}{2\beta} = \frac{1}{2}kT$$

> **结论**：在经典极限下，哈密顿量中每一个独立的平方项对系统平均能量的贡献恰好为 $\frac{1}{2}kT$。

# 7. 适用条件与局限

能量均分原理有两个关键前提，缺一不可：

1. **经典极限（高温）**：$kT \gg \Delta\varepsilon$，即热能远大于能级间距。此时能级近似连续，求和可替代为积分。振动自由度因为 $\Theta_v$ 很大（几百到几千 K），常温下不满足此条件，所以 $U_v^0 \approx 0$ 而非 $kT$。

2. **平方项形式**：该自由度必须在哈密顿量中以二次型出现。并非所有能量项都是平方项——例如相对论性能量 $E = \sqrt{p^2c^2 + m^2c^4}$ 就不是平方项，不能用均分原理直接处理。

# 8. 推导的本质直觉

从玻尔兹曼分布的角度看，**每个平方项就是一个独立的"能量容器"**。温度 $T$ 决定了有多少热能可以"灌入"这个容器。高斯积分的数学结构保证了每个平方项恰好分到 $\frac{1}{2}kT$——不多不少。这就是"均分"二字的深层含义。

这种"均分"只在能级连续的经典极限下成立；一旦量子效应显现（能级离散），均分原理就失效了——这正是 $C_V$ 在低温下偏离经典值的原因（例如 $H_2$ 的转动和振动在低温下被"冻结"）。
